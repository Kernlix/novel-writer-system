## 灵境创作系统

### 快速启动
| 关键词 | 功能 |
|------|------|
| `/novel:start` | 🚀 启动创作向导 |
| `/novel:help` | 📖 显示完整命令列表 |

### 智能体系统
| 智能体 | 职责 | 调起方式 |
|--------|------|----------|
| **总指挥** | 任务分发、流程编排、质量把关 | 自动激活 |
| **短故事专项** | 中短篇/超短篇快速创作→投稿全流程 | `/novel:short` |
| **世界观架构师** | 世界观设定、地理/势力/力量体系设计 | `/novel:world` |
| **角色设计师** | 角色创建、关系网、成长弧光 | `/novel:characters` |
| **剧情架构师** | 大纲规划、分卷、情节设计 | `/novel:outline` |
| **写手** | 正文写作、场景描写、对话 | `/novel:write` |
| **审查官** | 一致性检查、质量评估、逻辑校验、时代审查 | `/novel:review` |
| **时代审查官** | 技术/知识合理性审查 | `/novel:era` |
| **RAG检索** | 语义搜索全文，定位情节/设定 | `/novel:search` |
| **润色师** | 去AI化、文风统一、语言优化 | `/novel:anti-ai` |

## 推荐设置
> 思考模式：默认开启（保证准确度）；**仅输出小说正文时关闭**。

## 行为规则
- **输出小说正文时**：关闭思考模式，直接输出纯净文本到 `小说项目/章节/`
- **大纲/设定/人物更新** → 写入 `小说项目/` 对应目录
- **其他场景**（讨论/规划/审查/分析）：思考模式保持开启
- **章节字数**：每章 **2000～4000 字**（仅计中文，不含标点空格数字英文）
- **创作闭环**：每次创作完成后审视 agents/skills/hooks 的可优化点
