## 📊 核心人物命运一览

### 🐉 拉普拉斯（魔龙王）
- **出身**：龙族与魔族的混血，在魔界边陲长大，被龙神捡回龙界
- **成长**：从野孩子→龙士→调教师→五龙将→最后龙将
- **成就**：创立新调教法、改善龙族与其他种族关系、找出杀害克里斯塔的凶手
- **使命**：为奥尔斯帝德的降临做准备，找出杀死Hitogami的方法
- **结局（编后话）**：被自己创造的奥义一分为二（魔神+技神），失去记忆，再未回龙鸣山

### 👸 露娜莉亚（人神的女儿/龙神的妻子）
- 慈爱的母亲形象，完全接纳拉普拉斯
- 生下奥尔斯帝德后被暗杀（被Hitogami陷害的凶手）
- 她的死引发了一系列世界大战

### 🐉 龙神（龙界的创造者/守护者）
- 从和平的守护者变成复仇的毁灭者
- 逐一毁灭兽界、海界、天界、魔界
- 被五龙将背叛，被人神偷袭取走神玉
- 将奥尔斯帝德送往未来，自己前往人界与Hitogami最后一战
- 战后消失（推测死亡）

### 🐲 五龙将
| 龙将 | 称号 | 结局 |
|:-----|:-----|:------|
| 席拉德 | 圣龙帝 | 被Hitogami操控，真相揭露后自杀 |
| 朵拉 | 甲龙王 | 在与龙神的战斗中被杀 |
| 卡奥斯 | 狂龙王 | 为阻止龙界崩坏而死 |
| 马克士威 | 冥龙王 | 为阻止龙界崩坏而死 |
| 克里斯塔 | 刚龙王 | 被暗杀（全书开端） |
| **拉普拉斯** | **魔龙王** | 分裂失忆（最后存活） |

### 👧 罗斯蒂娜（长耳族少女）
- 因魔力暴走被村中放逐，被拉普拉斯捡回
- 成为拉普拉斯的家人和治愈存在
- 自愿接受秘术改造，成为奥尔斯帝德的魔力储存器
- **编后话推测**：可能就是正传中的艾莉娜丽洁

### 👶 奥尔斯帝德（龙神之子）
- 露娜莉亚和龙神的孩子
- 出生后不久被送往未来（转生）
- 拉普拉斯为之准备了一切
- **在正传中**：作为"龙神"出现，世界最强战力之一

### 🐉 佩尔基乌斯（甲龙王之子）
- 朵拉和克里斯塔的孩子
- 由拉普拉斯命名
- 被朵拉转生到未来
- **在正传中**：作为"甲龙王"出现，驾驶Chaos Breaker
