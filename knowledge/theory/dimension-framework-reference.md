# 维度分析框架参考

> 学习新作品时，必须先确定维度分析框架，再进行章节分析。

## 通用维度（每个作品必含）

| 维度 | 分析重点 |
|:-----|:---------|
| 👤 人物设定与成长 | 角色塑造、成长弧线、关系发展 |
| ⏰ 时间线 | 剧情时序、时间跨度、节奏把控 |
| 📋 作品大纲 | 整体结构、阶段划分、主题递进 |
| 🏠 场景构建 | 空间布局、感官锚点、环境描写 |
| ✒️ 文风语言 | 叙事风格、对话特色、修辞手法 |

## 专项维度（根据作品类型新增）

### 喜剧/异世界（9维度）
| 维度 | 分析重点 |
|:-----|:---------|
| 🎮 生存机制 | 生存压力、资源获取、危机应对 |
| 💬 社交互动 | 人际关系、阵营划分、信任与背叛 |
| 🔍 探索发现 | 世界观揭示、秘密发现、信息获取 |
| ⚔️ 战斗对抗 | 战斗设计、力量展示、策略运用 |
| 👑 支配权力 | 权力关系、地位变化、影响力 |
| 💰 获取创造 | 资源积累、能力建设、创造发明 |
| 😂 幽默元素 | 笑点设计、喜剧节奏、吐槽互动 |
| 💕 感情发展 | 感情线推进、互动甜蜜度、关系突破 |
| 🌍 世界构建 | 设定细节、文化差异、规则体系 |

### 悬疑/推理（7维度）
| 维度 | 分析重点 |
|:-----|:---------|
| 🎯 悬念设计 | 核心悬念、悬念维护、悬念释放 |
| 🔍 信息控制 | 读者知情度、角色知情度、信息不对称 |
| 🔗 伏笔网络 | 线索埋设、线索回收、红鲱鱼布局 |
| ⚡ 节奏张力 | 紧张-舒缓交替、高潮分布、翻页动力 |
| 🔄 反转结构 | 反转数量、反转层次、反转可信度 |
| 👁️ 叙事视角 | 视角选择、不可靠叙述者、视角切换效果 |
| 🧩 逻辑自洽 | 线索闭合度、逻辑漏洞、巧合合理性 |

### 死亡游戏/无限流（11维度）
| 维度 | 分析重点 |
|:-----|:---------|
| 🎲 死亡游戏机制 | 规则设计、胜负条件、规则漏洞利用 |
| ⚡ 超自然能力体系 | 觉醒条件、能力类型、限制与代价 |
| 🧠 心理博弈 | 信任与背叛、联盟与操控、智力对决 |
| 💀 人性探讨 | 极端环境下的道德选择、人物哲学 |
| 🔮 悬念与伏笔 | 隐藏真相、多重反转、伏笔埋设 |
| 🌍 世界观架构 | "终焉之地"规则、造神体系、更大格局 |
| 👤 人物设定与成长 | 角色塑造、成长弧线、群像刻画 |
| ⏰ 时间线 | 剧情时序、时间跨度、节奏把控 |
| 📋 作品大纲 | 整体结构、阶段划分、主题递进 |
| 🏠 场景构建 | 游戏空间、环境描写、氛围营造 |
| ✒️ 文风语言 | 叙事风格、对话特色、修辞手法 |

## 新作品维度设计流程

1. **分析作品类型**：确定是喜剧、悬疑、死亡游戏还是其他类型
2. **选择对应框架**：从上述框架中选择或组合
3. **补充专项维度**：根据作品特征新增特有维度
4. **锁定框架**：输出维度框架文档，本作品所有章节分析必须使用同一套维度
5. **框架迭代**：分析过程中发现维度不足时，可在后续批次补充（需记录变更）

## 注意事项

- 通用维度是**强制要求**，不能省略
- 专项维度是**推荐框架**，可根据作品特征调整
- 同一作品的所有章节必须使用**同一套维度**，保证可比性
- 维度数量控制在**7-11个**，过多会导致分析冗长
