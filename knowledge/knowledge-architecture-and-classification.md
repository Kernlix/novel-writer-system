# 灵境知识库架构与分类规则

> 本文档包含两部分：**架构**（知识存在哪）和**分类**（新知识放哪）。

## 一、知识库架构

### 分层原则

```
灵境系统（通用知识）
  ├── 标点用法、写作技巧、类型模板等
  └── 所有小说项目共享，不随具体小说变化

小说项目（专项知识）
  ├── 角色知识边界、人物档案、大纲、设定集
  ├── 章节正文、摘要、审查报告
  └── 仅当前小说使用，不同小说各自独立
```

### 通用知识库（灵境系统内）

路径：`knowledge/`

| 文件 | 内容 | 适用范围 |
|:----|:----|:---------|
| `punctuation-guide.md` | 标点符号用法规范 | 所有小说 |
| `writing-craft-enhanced.md` | 写作技法大全 | 所有小说 |
| `genre-templates.md` | 类型小说模板 | 按需加载 |
| `knowledge-architecture-and-classification.md` | 本文件 | 系统自身 |

### 专项知识库（小说项目内）

路径：`小说项目/知识边界/`

| 文件 | 内容 |
|:----|:----|
| `character-knowledge-boundary.md` | 角色知识边界地图 |

其他专项文件直接放在小说项目对应目录：
- `人物/` — 角色档案
- `大纲/` — 总纲、卷纲、伏笔追踪
- `设定集/` — 世界观、力量体系
- `章节/` — 正文

### 写作时查询规则

```
写前准备时：
  1. 通用写法规范 → knowledge/punctuation-guide.md
  2. 通用写作技巧 → knowledge/writing-craft-enhanced.md
  3. 小说专有设定 → 小说项目/大纲/、小说项目/设定集/
  4. 角色知识边界 → 小说项目/知识边界/character-knowledge-boundary.md

产生新知识时：
  1. 判断是通用还是专项 → 参考下方的分类规则
  2. 自动放入对应目录，无需询问
```

## 二、知识自动分类规则

> 每次产生新的知识沉淀时，按以下规则自动判断归属位置。

### 分类判断树

```
新知识产生
  │
  ├─ 是否与具体小说的角色/剧情/设定绑定？
  │   ├─ 是 → 放入当前小说项目
  │   │    ├─ 角色知道/不知道什么 → 小说项目/知识边界/
  │   │    ├─ 角色能力/外貌/关系 → 小说项目/人物/
  │   │    ├─ 剧情走向/伏笔 → 小说项目/大纲/
  │   │    ├─ 世界观规则 → 小说项目/设定集/
  │   │    └─ 已写正文/摘要 → 小说项目/章节/、章节摘要/
  │   │
  │   └─ 否 → 放入灵境系统
  │        ├─ 写作技法/标点/语法 → knowledge/
  │        ├─ 时代背景/历史知识 → .era-knowledge/
  │        ├─ 写作流程/系统机制 → skills/、hooks/
  │        └─ 智能体提示词优化 → agents/
  │
  └─ 不确定时问自己：换一本小说，这条知识还用得上吗？
       ├─ 用得上 → 灵境系统
       └─ 用不上 → 小说项目
```

### 快速判断表

| 知识类型 | 归属 | 示例 |
|:--------|:----|:-----|
| 标点符号用法 | 灵境 `knowledge/` | 句号不能断裂动作与感知内容 |
| 网文写作技巧 | 灵境 `knowledge/` | 爽点设计、钩子类型、节奏控制 |
| 时代背景/历史 | 灵境 `.era-knowledge/` | 中世纪农业技术、封建制度 |
| 写作流程规范 | 灵境 `hooks/`、`skills/` | pre-write流程、字数预算规划 |
| 智能体优化 | 灵境 `agents/` | 写手智能体质量标准更新 |
| **角色知识边界** | **小说项目 `知识边界/`** | 艾琳不知道系统存在 |
| **角色档案** | **小说项目 `人物/`** | 陈默腐化值12% |
| **剧情大纲** | **小说项目 `大纲/`** | 第二卷暗流初涌 |
| **世界观设定** | **小说项目 `设定集/`** | 力量体系、国家关系 |
| **章节正文/摘要** | **小说项目 `章节/`、`章节摘要/`** | 第72章空壳的行动 |
| **审查报告** | **小说项目 `审查报告/`** | chapter_072.review.json |

### 常见误区

| 错误做法 | 正确做法 |
|:--------|:---------|
| 把角色腐化值记在灵境系统 | 记在小说项目 `人物/` 对应角色档案 |
| 把标点规范记在小说项目 | 记在灵境系统 `knowledge/` |
| 把章节正文放在灵境系统 | 放在小说项目 `章节/` |
| 把写作技巧记在小说项目 | 记在灵境系统 `knowledge/` |
