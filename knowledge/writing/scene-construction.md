# 场景构建知识库

> 本库为写手智能体提供场景设计的方法论支持，涵盖场景类型、过渡技法、字数规划与锚点设计。

---

## 一、场景类型库

### 1. 室内对话场景

| 属性 | 说明 |
|------|------|
| **核心功能** | 推进剧情、揭示信息、角色塑造 |
| **感官分配** | 听觉（70%）：对话语气、环境音（壁炉/雨声）；视觉（20%）：表情、手势、环境细节；触觉（10%）：茶杯温度、座椅触感 |
| **节奏建议** | 中速偏慢。对话间可插入动作描写和内心独白作为节奏调节 |
| **字数参考** | 300～800 字（2～4 人对话体）；单人长独白 200～500 字 |
| **输出典型** | 密谋/情报交换/哲思辩论/情感摊牌 |

### 2. 室外探索场景

| 属性 | 说明 |
|------|------|
| **核心功能** | 展示世界观、渲染氛围、引导偶发事件 |
| **感官分配** | 视觉（50%）：环境描写、光影变化；听觉（20%）：风声/脚步/远景人声；嗅觉（15%）：草木/泥土/血腥；触觉（15%）：风/温度/地形 |
| **节奏建议** | 慢—中。用细节填充节奏，偶遇事件时短暂提速 |
| **字数参考** | 400～700 字 |
| **输出典型** | 进入新区域/日常巡逻/追踪搜索 |

### 3. 战斗/冲突场景

| 属性 | 说明 |
|------|------|
| **核心功能** | 制造张力、展示能力、催化成长 |
| **感官分配** | 听觉（35%）：撞击/嘶吼/兵器破空；视觉（40%）：动作快切/血迹/特效；触觉（20%）：冲击/疼痛/肌肉发力；味觉（5%）：血腥味/汗味 |
| **节奏建议** | 快速。短句、段落密集，动作之间可用半秒停顿制造节奏 |
| **字数参考** | 300～600 字（单回合）；600～1200 字（多回合完整战斗） |
| **输出典型** | 遭遇战/决斗/群战/追逐战 |

### 4. 过渡场景（地点转移）

| 属性 | 说明 |
|------|------|
| **核心功能** | 衔接剧情、调节节奏、提供思考空间 |
| **感官分配** | 视觉（40%）：快速变化的环境剪影；听觉（30%）：交通工具声/脚步声；内心（30%）：角色思考/回忆/计划 |
| **节奏建议** | 快—中。不宜过多描写，点到即止 |
| **字数参考** | 100～300 字（简单转移）；300～500 字（含沿途事件的复杂转移） |
| **输出典型** | 赶路/乘坐交通工具/穿越大厅/上楼 |

## 二、场景过渡技法

### 2.1 时间跳转

**表示方式：**
```
★ ★ ★
```

```
---
```

**过渡句示例：**
- 「当陈默再次睁开眼时，窗外的天光已经变成了暮色。」
- 「三个小时后，莉莉丝终于从废墟中走了出来。」
- 「这一夜，陈默睡得并不安稳。——翌日清晨，———」

### 2.2 空间转移

**表示方式：**
```
★ ★ ★
```

**过渡句示例：**
- 「与此同时，城堡另一端的密室里——」
- 「视线从深渊裂隙移开，回到营地的篝火旁。」
- 「同一时刻，不同地点：」

### 2.3 视角切换

**表示方式：**
```
---
```

**过渡句示例：**
- 「他不知道的是，在他转身离开之后——」
- 「从莉莉丝的视角看去，陈默的背影显得格外疲惫。」
- 「如果此时有人从高空俯瞰，就会发现——」

### 2.4 过渡技巧要点

| 技法 | 适用场景 | 注意事项 |
|------|----------|----------|
| 硬切（★ ★ ★） | 大幅度时间/空间跳转 | 确保读者能快速理解新的时间/空间位置 |
| 软过渡（过渡句） | 小范围转场 | 用一句承上启下的句子自然衔接 |
| 平行蒙太奇 | 多线叙事 | 在 A 线关键节点切到 B 线，保持悬念 |
| 回忆闪回 | 过去事件插入 | 明确标记时间点，用斜体或特殊格式区分 |

## 三、每章场景规划

### 3.1 场景数量建议

每章建议 **3～5 个场景**，按本章功能分配：

| 本章功能 | 场景数 | 搭配建议 |
|----------|--------|----------|
| 推进主线 | 4～5 | 1 过渡 + 2 对话 + 1～2 冲突/探索 |
| 角色塑造 | 3～4 | 1～2 对话 + 1 内心独白 + 1 过渡 |
| 世界观展示 | 3～4 | 2～3 探索 + 1 对话 |
| 高潮/大事件 | 4～5 | 1 过渡 + 1 室内预备 + 2～3 战斗/冲突 |

### 3.2 字数预算分配

总字数目标：**2000～4000 字（仅中文）**

按 3～5 个场景分配示例：

```
总字数 2500 字 → 3 个场景：
  场景1（过渡/铺垫）： 300～500 字
  场景2（核心事件）： 1200～1500 字
  场景3（收尾/钩子）： 500～700 字

总字数 3500 字 → 4 个场景：
  场景1（开局/过渡）： 400～600 字
  场景2（发展/对话）： 1000～1200 字
  场景3（转折/冲突）： 800～1000 字
  场景4（收尾/悬念）： 500～700 字

总字数 3000 字 → 5 个场景：
  场景1（过渡）：      200～300 字
  场景2（对话/铺垫）：  600～800 字
  场景3（探索/小冲突）：600～800 字
  场景4（高潮）：      600～800 字
  场景5（收尾/钩子）：  400～500 字
```

### 3.3 场景逻辑递进

每个场景应承担明确功能，场景之间形成逻辑链：

```
场景1：设定前提（地点/时间/情绪基调）
   ↓
场景2：引入冲突/信息（对话/发现）
   ↓
场景3：冲突升级/转折（战斗/意外/揭示）
   ↓
场景4：角色反应/决策（内心戏/对话）
   ↓
场景5：收尾+下章钩子（悬念/新方向）
```

**递进质量检测：**
- 删除任意一个场景，逻辑链是否断裂？
- 场景顺序是否可以互换？如果可以，说明缺乏逻辑依赖
- 每个场景是否至少推动以下一项：情节/角色/世界观/悬念

## 四、场景锚点

### 4.1 锚点设计原则

每个场景选取 **1 个核心感官锚点** + **1 个辅助情感锚点**，让读者形成记忆点。

### 4.2 感官锚点类型

| 感官 | 锚点示例 |
|------|----------|
| **视觉** | 墙上斑驳的血迹、摇曳的烛影、对方眼中倒映的紫光、雨幕中的孤塔 |
| **听觉** | 滴水声的节奏、脚步声的回响、武器的嗡鸣、远处若隐若现的钟声 |
| **触觉** | 金属剑柄的冰冷、石头地面的潮湿、绷带下伤口的刺痛、腐化值上升时的灼烧感 |
| **嗅觉** | 铁锈味（血腥）、发霉的空气、腐烂的木头、深渊的硫磺味 |
| **味觉** | 嘴里的血腥味、劣质麦酒的苦涩、雨后空气的甜润 |

### 4.3 锚点搭配策略

```
强烈情感场景 → 触觉锚点（疼痛/灼烧/冰冷）+ 听觉锚点
悬疑氛围场景 → 听觉锚点（滴水/脚步声）+ 视觉锚点（阴影/残影）
温情/日常场景 → 嗅觉锚点（食物/草木）+ 视觉锚点（暖光）
宏大世界观场景 → 视觉锚点（全景/远景）+ 触觉锚点（风/震动）
```

### 4.4 锚点复用技巧

- 同一锚点在不同章节出现，形成序列记忆（如某扇门的吱呀声）
- 关键锚点在角色情绪变化时变体（门的吱呀声从日常 → 刺耳 → 消失）
- 锚点与角色绑定（某角色出场总伴随特定气味/声音）

## 附：场景构建检查清单

写作前逐项确认：
- [ ] 本章选用了几种场景类型？搭配是否合理？
- [ ] 场景之间过渡方式是什么？（硬切/软过渡/平行蒙太奇）
- [ ] 每个场景的字数是否在建议范围内？总和 ≥ 2000？
- [ ] 场景间是否有逻辑递进关系？
- [ ] 每个场景是否分配了至少一个感官锚点？
- [ ] 是否有复用已在前文建立的锚点以形成记忆序列？
