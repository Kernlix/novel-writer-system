# 📋 先例推理战略叙事 Skill

> **核心来源**：《十日终焉》第516章（齐夏通过钱五杀死生肖未引神兽的先例，推导出可以杀死神兽）
> **提炼方法**：从策略与博弈、游戏规则设计维度中提取可复用技法
> **适用场景**：需要说服团队执行高风险计划时；死亡游戏/无限流作品中制定新战略时

---

## 技法1：先例推理法 (Precedent Reasoning)

### 1.1 核心原则

通过分析已有实践（先例）推导出新策略。先例比理论更能说服团队冒险——「有人做过」比「理论上可行」更有说服力。

### 1.2 设计步骤

1. **建立先例**：角色A分享一个过去的实践（钱五用「双生花」杀死了面试生肖）
2. **分析先例**：从先例中提取关键信息（双方同时死亡时，神兽不干预）
3. **推导新策略**：基于先例推导出新的可能性（可以杀死神兽）
4. **质疑与确认**：团队成员质疑先例的适用性，最终确认策略可行

### 1.3 示例

```
钱五的先例：
「我用『双生花』和面试生肖人马同归于尽过，但没有引来神兽。」

齐夏的推论：
「如果杀死生肖时双方同时死亡，神兽不干预……那杀死神兽是否也可行？」

→ 先例提供了「可能性」
→ 推论将可能性升级为「策略」
→ 团队基于先例+推论制定行动计划
```

### 1.4 注意事项

- 先例必须是「真实的」——不能是角色的猜测或道听途说
- 先例的适用性必须有「不确定性」——否则就不叫冒险
- 先例推理后需要有「补充信息」——先例只是起点，不是终点

## 技法2：三层博弈设计 (Three-Layer Game Design)

### 2.1 核心原则

齐夏的博弈是多层的：①评估能否杀死玄武（能力对抗）②寻找读心者（情报战）③排查极道渗透（组织安全）。三层博弈同时运作，展现战略家的思维深度。

### 2.2 设计步骤

1. **第一层：能力对抗**——评估直接对抗的可行性（能否杀死不灭的玄武）
2. **第二层：情报战**——寻找关键信息（必须找到「读心」持有者）
3. **第三层：组织安全**——排查内部威胁（极道是否混入了猫队）
4. **三层同时推进**：在同一次会议/对话中推进三层博弈

### 2.3 注意事项

- 三层博弈必须有逻辑关联——不是随机的三个问题
- 每层博弈的「回答」为下一层提供信息
- 三层博弈的节奏：第一层快速确认→第二层深入讨论→第三层在最后时刻抛出

## 技法3：缄默会议封闭叙事 (Sealed Meeting Narrative)

### 3.1 核心原则

用超自然能力为会议提供物理密封，既合理化了「可以谈论机密」的前提，又通过「缄默即将结束」制造时间压力。

### 3.2 设计步骤

1. **设置密封条件**：云十九的「缄默」能力让会议内容无法被外界听到
2. **合理化机密谈话**：因为有密封，角色可以谈论最敏感的话题
3. **设置时间限制**：缄默能力即将结束——时间压力让讨论更紧迫
4. **时间控制精准**：齐夏在缄默结束前才提出最敏感的问题（极道疑云）

### 3.3 注意事项

- 密封条件必须是「超自然的」——物理锁门不够，需要有「被发现」的风险
- 时间限制必须是真实的——角色知道缄默什么时候结束
- 最敏感的问题放在最后——利用时间压力制造紧迫感
