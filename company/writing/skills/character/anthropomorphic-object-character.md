# 神器拟人化角色写作 Skill

> 来源：KonoSuba第8卷缺口分析。阿吉斯（说话的铠甲）被赋予性偏好、自我意志、支配反转。

## 技法A：无机物的完整人格

阿吉斯不只是"会说话的道具"，它有性偏好（选女性主人）、自我意志（拒绝不喜欢的持有者）、感情（崇拜惠惠到宗教狂喜）。让读者忘记它是铠甲，把它当完整的角色对待。

## 技法B：支配反转

主人→神器关系：正常是人类支配神器。阿吉斯的选择权让这个关系反转——神器在挑选主人，被选中的人类反而有了"被选中"的卑微感。

## 技法C：拟人化的喜剧用途

无机物表现出"过于人类"的特质（性偏好/吃醋/要求/嘴硬）是天然的喜剧引擎。阿吉斯对惠惠的狂热崇拜、对和真的鄙视、对处女的执着——每个特征都是一幕喜剧。

## 协作规则
| 关系 | 说明 |
|:-----|:------|
| humor-writer | 主归属，拟人化喜剧引擎 |
| worldbuilding | 神器的设定规则（什么样的人才能使用、限制条件） |
| scene-construction | 装备/道具的出场场景构建 |
