# 📖 写作技法知识库

> 从优秀网文中提炼的专业写作技法。

---

## 零、基础技巧

### 场景写作
- 五感描写：视/听/触/嗅/味，每场景至少 3 种
- 具体胜过抽象：不说"很美"说"阳光把她的睫毛染成金色"
- 动态胜过静态：用动作推动场景

### 情节设计原则
- 冲突是故事的引擎
- 每一章都要回答"为什么读者会在意"
- 好的情节 = 角色 × 决定 × 后果

### 对话基本要点
- 每句对话应有功能（塑造角色/推进情节/提供信息）
- 角色说话方式要能区分
- 沉默有时比对话更有力

### 节奏控制基础
- 紧张段落用短句，抒情段落用长句
- 动作场面节奏快，情感场面节奏慢
- 每 3-5 章安排一个小高潮

## 一、内面独白技法

### 什么是内面独白
角色在内心与自己的对话，是网文区别于传统文学的重要特征。

### 三种独白类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **反省型** | 回顾过往，自我剖析 | 「我到底做了什么？……」 |
| **吐槽型** | 对当前情境的内心评价 | （对着严肃场景内心吐槽） |
| **决策型** | 在关键时刻的内心权衡 | 「该怎么做？冲上去还是撤退？」 |

### 写作要点
- 不要直接说情绪，用独白展现角色性格
- 独白要口语化，符合角色身份
- 独白不要太长（100-300字为宜），否则变成说教
- 在动作/对话的间隙插入独白，而非一整段纯独白

## 二、开篇钩子设计

### 黄金 200 字法则
读者在前 200 字内决定是否继续阅读。

### 五大钩子类型

#### 1. 身份反差钩
制造角色身份与处境的强烈反差。
> 「本人现年三十四岁，居所不定也没有职业。」

#### 2. 情境紧迫钩
把读者直接扔进紧张场景。
> 暴雨、流落街頭、身无分文。

#### 3. 悬念开场钩
抛出无法忽视的问题。
> 「那扇门后面有什么，没有人知道。——直到那天。」

#### 4. 金句哲理钩
以一句有意境的话开场。
> 「人生最讽刺的事，就是你想要的东西往往在你放弃之后才出现。」

#### 5. 日常破裂钩
从最普通的日常开始，突然打破。
> 「那是一个普通的下午。——如果那辆车没有失控的话。」

## 三、情感表达技法

### Show, Don't Tell 进阶版

| Tell（AI味） | Show（网文味） |
|-------------|---------------|
| 他很生气 | 他指节发白，把杯子捏得咯咯作响 |
| 她很开心 | 她哼着歌，步子比平时轻快了许多 |
| 气氛很尴尬 | 三个人盯着茶杯，蒸汽在空气中缓缓上升 |
| 他感到绝望 | 他靠在墙上，盯着天花板裂缝 |

### 情感暗示技巧
- **重复动作**：反复做同一件小事（沏茶、整理衣角）暗示内心不安
- **反常行为**：平时开朗的人突然沉默，比描写悲伤更有力
- **环境映衬**：心情好时雨声是节奏，心情差时雨声是烦扰

## 四、对话写作技法

### 对话 + 动作的自然融合

```
错误：「你要去哪里？」他说。
正确：「你要去哪里？」他抓起外套，手指在袖口停顿了一瞬。
```

### 对话节奏控制
- **紧张时**：短句、少修饰、快节奏
- **放松时**：长句、多细节、慢节奏
- **沉默的力量**：有时不回答比回答更有力

### 角色语言差异化
每个角色应有独特的说话方式：
- **语调**：温和/尖锐/沉稳/轻浮
- **用词**：正式/随意/方言/口癖
- **句式**：简洁/啰嗦/反问/命令
- **习惯**：口头禅、停顿方式、笑的方式

### 对话避坑清单（灵境系统实战沉淀）

| # | 陷阱 | 错误表现 | 正确做法 |
|:-:|:-----|:---------|:---------|
| 1 | **多轮冗余问答** | 「下矿了？」「下了。」「和铁锤一起？」「对。」 | 合并：「下矿了？和铁锤一起？」「下了。」——减少机械问答 |
| 2 | **对话＝信息搬运** | 「铁锤怎么说？」「半年。」——对话纯传递事实，没有情绪 | 沉默、叹气、停下手上的动作后再回答——让对话本身承载情绪 |
| 3 | **关键台词太轻** | 前面铺垫了"张嘴又合上""手抖了三秒"，但说出来的话跟日常没区别 | 关键台词要能撑起前面的铺垫。如果话不够重，宁可减少铺垫 |
| 4 | **叙事替对话干活** | 叙事说"他很愤怒"但对话内容平淡——情绪全在旁白里 | 让对话本身传递情绪。读者能从"你再说一遍？"感受到愤怒，不需要旁白解释 |
| 5 | **所有角色说一样的话** | 遮住角色名猜不出谁在说话——铁锤和莉莉丝的区别消失了 | 铁锤的短、狼牙的野、莉莉丝的懒洋洋、老托马斯的隐忍

## 五、世界观自然融入

### 融入时机

| 时机 | 方法 | 示例 |
|------|------|------|
| **战斗中** | 通过威力展示等级体系 | 「这一剑劈开岩石——至少上级实力」 |
| **教学时** | 师生对话中自然带出 | 「记住，魔力需要从心脏引导到指尖」 |
| **购物时** | 价格展示经济体系 | 「一把铁剑要5个银币？抢钱啊！」 |
| **闲聊中** | 角色聊天透露背景 | 「你知道北方的魔王换了人吗？」 |

### 禁忌
- ❌ 单独一段「设定说明」
- ❌ 多个新概念同时抛出
- ❌ 用读者已知视角解释常识

## 六、节奏控制技法

### 章节内节奏
```
叙事 ↗ 对话 ↗ 动作高潮 ↘ 独白反省 ↘ 平静结尾
```

### 卷内节奏
```
1-3章：铺垫建立
4-6章：冲突升级
7-9章：高潮爆发
10章：余韵/新线索
```

### 长系列节奏（10+卷）
- 每个阶段（3-5卷）要有明确的主题和目标
- 中期要安排「最大挫折」（主角跌入谷底）
- 后期逐步揭示伏线，回收伏笔

## 七、闲话与外传技法

### 闲话的作用
1. **填补时间线**：主角修炼/旅行时，配角的世界也在运转
2. **切换视角**：让读者看到主角看不到的关键事件
3. **调节节奏**：紧张主线后的情绪缓冲
4. **深化配角**：让配角从工具人变成有血有肉的角色

### 闲话插入时机
```
主线高潮后 → 闲话缓冲（1-2章） → 下一轮铺垫 → 主线推进
```

### 闲话篇幅
- 一般 1-3 章，不宜过长
- 过长的外传可以单独成册

## 八、成长弧光分阶段模型

### 七阶段模型

| 阶段 | 章节跨度 | 任务 |
|------|---------|------|
| 1. 起点 | 1-3章 | 展示角色缺陷和现状 |
| 2. 觉醒 | 3-5章 | 给予改变契机 |
| 3. 尝试 | 5-10章 | 首次努力，遭遇挫折 |
| 4. 突破 | 3-5章 | 关键事件迫使超越自我 |
| 5. 反复 | 5-8章 | 遇到困难，旧习复发 |
| 6. 蜕变 | 3-5章 | 真正成长，旧我消失 |
| 7. 终点 | 1-3章 | 达成终极变化 |

### 关键原则
- 每个阶段需跨越足够章节（不可一章完成成长）
- 成长路上必须有反复和退步
- 外部事件推动内部变化（事件 → 冲击 → 反思 → 改变）
