# 🔎 招募流程 Skill

## 命令
`/novel:recruit`

## 功能
从能力评估到新Agent集成的全流程。

## 工作流程
```
差距分析 (gap-analysis-agent)
  → 岗位设计 (job-designer-agent)
  → 技能研发 (skill-engineer-agent)
  → Agent集成 (agent-integrator-agent)
  → 通知用户
```

## 触发条件
- 用户反馈反复提到某种能力的不足
- 审核报告显示系统性质量短板
- 学习部门发现新的技法方向
- 负责人主动发起能力升级
