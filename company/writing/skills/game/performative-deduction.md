# 表演式推理（Performative Deduction）

> 让角色通过身体动作、模拟演示来展示推理过程，比口头解释更有视觉冲击力和说服力。

## 定义

用"表演"替代"陈述"来传达推理结论——让角色通过身体动作（如模拟握刀、演示死亡姿态、重建犯罪现场）来展示推理过程/结论，而不是通过口头解释。这是"Show, don't tell"原则在推理叙事中的极致应用。

## 适用场景

- 刑侦/推理/悬疑题材中的案件解密
- 需要让读者直观理解复杂的物理/空间关系时
- 死亡游戏/规则怪谈中的规则演示

## 技法要点

### 1. 表演替代陈述

核心原则：**做出来，而不是说出来**。

- 角色不应说"她是自杀的"，而应**表演**自杀的握刀动作
- 角色不应说"凶手是从窗户逃走的"，而应**演示**翻窗的体态和角度
- 用身体动作+空间关系来展示推理，让读者"看到"真相而非"听到"真相

### 2. 表演的三段式结构

1. **目标设定**：角色说出要验证/演示什么（如"刀刃朝上……")
2. **身体演示**：角色实际演示动作（如跪下、握刀、模拟刺入）
3. **结论确认**：从演示中推导出结论（如"她只可能是自杀"）

这三段应保持紧凑，演示部分是核心，其他部分是辅助。

### 3. 利用角色职业身份

- 赋予角色专业知识（刑警、法医、格斗专家），让表演自然可信
- 职业身份让角色有"表演推理"的资格，不显得做作
- 不同职业背景的角色会对同一事件有不同推理方式

### 4. 视觉反转

- 表演本身可以是一个**视觉反转点**——读者之前可能没有想到的握刀角度、空间关系
- 让推理的结论通过"看到了"而非"听到了"来传达
- 如果推理本身很精妙，视觉效果会放大其冲击力

## 示例对比

### ❌ 错误做法
```
"她应该是自杀的。"苏闪说。
"为什么？"齐夏问。
"因为刀刃朝上，如果是别人刺入，刀刃应该朝下。"
```
纯口述推理，缺乏视觉冲击力。

### ✅ 正确做法
```
苏闪从余念安的手中取出了那把水果刀。
"等一下，你看……她的手指是这么握着刀柄的。"
她跪在地上，手心朝上，将刀尖刺向自己的胸口。
"刀刃朝上……她想要自杀，于是她用了一整夜的时间等待着这一刻。"
```
通过身体动作的演示，读者"看到"了推理过程，感受远超口述。

## 与相关技法的区别

| 技法 | 核心 | 区别 |
|:-----|:-----|:------|
| 硬推理叙事 | 数学/逻辑谜题的叙事化 | 表演式推理用物理动作，硬推理用数字/逻辑 |
| 博弈场景可视化 | 旁观者解说博弈过程 | 表演式推理是角色亲自演示而非旁观者解说 |
| 过程暴力 | 暴力动作的过程而非结果 | 表演式推理是推理过程而非暴力过程 |

## 来源

- **作品**：《十日终焉》
- **示例章节**：第234章「杀人者是？」——苏闪演示"刀刃朝上"的自杀握刀姿态
- **示例分析**：通过表演式推理而非陈述式推理来传达案件真相——最高级的推理叙事

## 本能提取

> 推理的力量在"做出来"而不在"说出来"——身体动作传达的信息量远超语言。
