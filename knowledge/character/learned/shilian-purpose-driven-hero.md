# 目的驱动型 vs 自我厌恶型——主角原动力模型

## 技法分析

《十日终焉》的齐夏与《无职转生》的鲁迪乌斯形成了教科书式的对照——两个都是缺陷型主角，但底层驱动力完全不同，导致叙事引擎和读者体验截然相反。

**齐夏（目的驱动 / Purpose-Driven）：**
- 核心句式："我需要……"，"要达成……必须……"
- 有清晰的外在目标（找到余念安、破局、活下去）
- 极度理性：评估利弊、计算概率、牺牲可牺牲的
- 他不是"好人"，他是"做事的人"
- 决策逻辑：目标→路径→最优解→执行（中间省略情感环节）
- 读者的期待落点："他能不能达成目标？"

**鲁迪乌斯（自我厌恶驱动 / Shame-Driven）：**
- 核心句式："我不能……", "前世的我……"
- 动力来自逃离前世的自己（NEET/废柴/啃老）
- 情感驱动：恐惧重蹈覆辙、渴望被认可、补偿心理
- 他是"想变好但总搞砸的人"
- 决策逻辑：恐惧→冲动→搞砸→反思→成长（情感贯穿始终）
- 读者的期待落点："他能不能改变自己？"

| 对比维度 | 目的驱动（齐夏） | 自我厌恶驱动（鲁迪乌斯） |
|---------|---------------|---------------------|
| **动力方向** | 向外：达成某事 | 向内：不成为某种人 |
| **决策逻辑** | 利弊计算，可牺牲他人 | 情感驱动+补偿心理，怕失去 |
| **道德弹性** | 高——必要时可以做"错事" | 低——道德约束强，事后内疚 |
| **角色温度** | 冷——读者敬畏而非喜爱 | 暖——读者同情而非崇拜 |
| **读者期待** | "他能成功吗？" | "他能改变吗？" |
| **叙事张力源** | 外部阻力阻止目标达成 | 内心阻力阻止自我超越 |
| **成长弧特征** | 能力升级+策略进化 | 人格修复+价值观重塑 |

**为什么需要区分驱动力类型？**

两种驱动力产生两种完全不同的叙事引擎。目的驱动型主角的小说像一场"解题竞赛"——读者想看主角如何用智慧和策略突破层层障碍。自我厌恶驱动型主角的小说像一场"疗愈之旅"——读者想看主角如何接纳自己、超越过去。

这不是"好"与"坏"的区别，而是设计选择。关键在于：**在设计主角的初期阶段，必须明确其核心驱动力类型，并让后续所有情节设计服务于这个驱动力。**

## 与已有知识的对照分析

**对照 character-evolution.md：**
character-evolution.md的"角色弧光类型"一节列出了四种弧光（成长弧/悲剧弧/揭秘弧/堕落弧），但这四种弧光描述的是弧的**形状**（角色从A到B的变化轨迹），没有回答"是什么力量推动角色走完这段弧"的问题。驱动力模型提供了一个前置设计维度：在决定弧光形状之前，先决定弧光的发动机类型。

**对照 shilian-ability-personality-echo（已学）：**
能力与性格的绑定回答了"角色的能力从哪来"（能力=灵魂的呐喊），驱动力模型回答了"角色为什么动"（行动=驱力的外化）。二者互补但不重叠。

**对照 mushoku-flawed-pov-honesty（已学）：**
缺陷主角的诚实性叙事关注的是"怎么叙述缺陷主角的内心"，是叙事技术层面。驱动力模型关注的是"缺陷主角的发动机是什么"，是角色设计层面。鲁迪乌斯的自我厌恶是一种驱动力类型，十日终焉展示了另一种同样有效的类型。

## 可复用场景

- 新作品主角设计阶段——先确定驱动力类型再设计弧光
- 现有角色诊断——主角的行为是否出现了"动机不统一"的问题
- 对比教学——同类型作品（如无限流/智斗类）使用何种驱动力更有优势

## 自我提升方案

### 对照分析

提取当前小说主角在连续5章中的核心决策，分类统计：
- 目的驱动决策（"为了达成X，所以做Y"）占比多少？
- 情感/补偿驱动决策（"因为害怕/愧疚/冲动，所以做Y"）占比多少？
- 是否有决策动机模糊的情况？

### 优化措施

1. **驱动力宣誓**：在主角前3次出场中，至少有一次明确暗示/展示主角的核心驱动力（不一定要说出来，但要让读者能感觉到）
2. **决策一致性检查**：每次主角做重大决定时，反问"这个决定符合他的驱动力类型吗？如果换一种驱动力类型的主角，会做同样决定吗？"
3. **驱动力冲突设计**：如果想制造角色深度，可以在主要驱动力之下埋一个次级驱动力（如齐夏purpose-driven之下藏着对余念安的个人情感），让二者偶尔冲突

### 应用承诺

新建主角时，在角色档案中增加"核心驱动力类型"字段，写明驱动力类型、一句驱动力宣言、以及驱动力在决策中的典型表现模式。
