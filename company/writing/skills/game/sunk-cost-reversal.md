# 沉没成本反转叙事技法

> 「如果我得不到好结局，那我之前杀掉的人算什么？我付出的努力算什么？我每天按时上班的目的是什么？」——地狗·第399章

## 概述

在叙事中，角色的立场转变通常由三种驱动力驱动：**道德觉醒**（被说服）、**情感打动**（被感动）、**利益计算**（被说服但基于利益）。「沉没成本反转」属于第三种，但具有独特的叙事力量——它不是简单的「给更多钱我就加入」，而是「如果我不继续，之前的投入就全浪费了」。

这种反转的力量来自经济理性的冷酷美——角色不是变了，而是算清了账。

---

## 技法结构

### 核心公式

```
沉没成本（已投入的不可回收成本）
  + 新信息（这些成本可能无法回本）
  = 继续投入的强制理由（为了避免前功尽弃）
```

### 三要素

| 要素 | 说明 | 示例（地狗） |
|:-----|:------|:-------------|
| **已投入成本** | 角色为当前立场已付出的代价 | 杀掉的人数、工作的年限、小心翼翼的隐忍 |
| **新信息** | 揭示继续当前立场会导致成本"白费" | 「不能晋升天级」→ 所有努力无回报 |
| **反转决策** | 基于成本核算选择新立场 | 从反对造反变为支持造反 |

### 执行步骤

1. **建立角色的投入**：展示角色为当前系统/立场投入了什么（时间、血汗、道德成本）
2. **引入威胁信息**：给角色一个「继续投入可能全部白费」的新信息
3. **展示计算过程**：角色重新计算收益和成本（不是道德评估，是经济评估）
4. **触发反转**：角色基于「避免沉没成本浪费」做出新的选择

| 场景类型 | 适用度 | 说明 |
|:---------|:------|:------|
| 造反/叛乱情节 | ⭐⭐⭐⭐⭐ | "我已经杀了这么多人，我不能白干" |
| 角色黑化 | ⭐⭐⭐⭐⭐ | "我已经越界了，回头只会让之前的越界变得毫无意义" |
| 反派招募 | ⭐⭐⭐⭐ | 用"你已经没有退路了"迫使对方继续 |
| 道德困境 | ⭐⭐⭐⭐ | 犯下不可挽回的错误后，继续还是停止？ |
| 职场/体制批判 | ⭐⭐⭐⭐ | "我都干了这么多年了，现在放弃就全白费了" |

## 配合指南

### 前置条件
- 角色必须有可量化的「投入」（杀人次数、劳动年数、金钱、道德成本）
- 需要一个「成本即将白费」的触发信息
- 计算过程必须展示（不能暗中完成），让读者看到角色的推理

### 避坑指南
| 陷阱 | 说明 |
|:-----|:------|
| **突然计算** | 角色之前从未计算过成本，突然开始计算——需要前置铺垫 |
| **纯理性角色** | 只有理性角色适合沉没成本反转——冲动型角色需要用其他驱动力 |
| **忽视情感成本** | 沉没成本不只是金钱——道德成本、时间成本、人际关系成本都可以 |
| **反转后无变化** | 反转后角色的行为模式需要相应改变——不能"换了立场但动作一样" |

### 与其他反转类型的区别

| 特征 | 道德觉醒 | 情感打动 | 沉没成本反转 |
|:-----|:---------|:---------|:-------------|
| 驱动力 | "这是对的" | "我心疼你" | "我不能白干" |
| 说服路径 | 价值论证 | 情感共鸣 | 经济核算 |
| 读者共鸣 | 理性认同 | 感性共鸣 | 冷酷的震撼 |
| 角色一致性 | 可能突然变好 | 可能戏剧化 | 前后最一致（因为价值体系未变） |
