## 一、灯系列概述

**灯系列**（《为美好的世界献上祝福！》达斯特外传）是素晴本传的平行外传，以**达斯特（Dust）**——阿克塞尔最底层冒险者、和真小队的「编外损友」——为主视角的搞笑冒险故事。

| 属性 | 内容 |
|:-----|:------|
| 原作 | 暁なつめ（KonoSuba 本传） |
| 外传作者 | 昼熊 |
| 卷数 | 全7卷 |
| 时间线 | 与本传平行，视角聚焦边缘角色 |
| 分析框架 | 7维度本能分析（各卷维度框架略有调整） |

### 全卷弧光

达斯特从"逃避过去的亡命龙骑士"到"为守护之物挺身而出的城镇英雄"，全7卷完成了完整的角色弧光：

- **灯1**：底层混混日常——认知错位与三重知晓结构
- **灯2**：缺陷永动机——达斯特的贪婪+好色+冲动三重缺陷驱动笑点
- **灯3**：生存本能——安乐少女事件、监禁麻木、逃跑本能
- **灯4**：双线并行——泰勒的春天×公主的婚约，欺骗与信任
- **灯5**：身份反转——达斯特=前龙骑士莱因·薛克、琳=公主化身
- **灯6**：双层交叉叙事——过去（龙骑士×公主）与现在（莉欧娜尔假冒琳）
- **灯7**：最终决战——从流放到归属，龙骑士回归阿克塞尔

> *注：灯1-4的部分维度命名差异较大（如灯1的「成长」维度在灯5-7被独立细化）

### 缺口识别

1. **格式不统一**: 灯1使用 "主题/人物/情节/叙事/语言/喜剧/价值"，灯2使用 "缺陷体系"，灯3使用 "7大生存本能体系"，灯5使用 "7Agent"——各卷没有统一的维度命名方案
2. **JSON格式仅有灯1**: 其他卷没有结构化数据导出，不利于RAG检索
3. **本能密度递减**: 灯1 ~108条 → 灯7 ~30条，分析细致度逐卷下降
4. **跨卷关联缺失**: 本能ID未实现跨卷交叉引用，例如达斯特的「逃跑本能」（灯3）与「龙骑士身份」（灯5）没有建立关联
