# 剧情架构师智能体专用知识库

> 本知识库可在每次剧情更新后根据经验优化更新。

## 伏笔管理规范

### 伏笔追踪表格式
```
| 编号 | 伏笔描述 | 埋设章节 | 预期回收 | 优先级 | 状态 |
```

### 优先级分级
| 级别 | 说明 | 处理时限 |
|:----|:-----|:---------|
| ★★★★★ | 全书级核心谜题 | 需要跨卷规划 |
| ★★★★ | 本卷级关键伏笔 | 卷内回收 |
| ★★★ | 角色/情节线伏笔 | 1~2卷内回收 |
| ★★ | 细节/彩蛋级 | 可长期放置 |

### 状态标记
| 状态 | 含义 | 操作 |
|:----|:-----|:-----|
| 待回收 | 已埋设未回收 | 等待合适章节 |
| 进展中 | 已部分展开 | 持续跟踪 |
| 已回收 | 已揭示/解决 | 移入已回收表 |

## 节奏控制

### 张弛节奏（每5章一个周期）
```
紧张章 → 紧张章 → 缓和章 → 爆发章 → 余韵章
示例：
72(紧张) → 73(爆发·罗姆死) → 74(余韵·葬礼) → 75(紧张·子爵) → 76(爆发·子爵终局)
```

### 爽点分布密度
- 本卷（第2卷）主要爽点间隔：每5~8章一个高潮
- 刀片（情感打击）间隔：每8~12章一个
- 暗线揭示间隔：每3~5章一个小揭示

## 节奏模式扩展

### 【素晴学】反高潮节奏模式
不是所有高潮都必须"冲天"。可以选用的替代节奏模式：
- **反高潮模式**：建立紧张→荒诞/意外解决→回落到日常。适用于喜剧成分高的作品或需要打破"必然成功"预期的章节
- **限制爆发模式**：限制积累→在限制内找到创意方案→限制升级（非限制解除）。适用于不希望角色线性变强的成长型作品
- **日常引擎模式**：经济/社交压力积累→被迫行动→小胜利/小失败→新的日常压力。适用于日常系/慢热型作品

### 【素晴学】卷级循环结构模板
借鉴素晴的独立卷循环：
1. **开篇日常锚点**：展示主角当前状况（非战斗状态），暗示经济/社交需求
2. **触发事件（非天降）**：事件从角色日常困境中自然生长出来（如缺钱→接任务→陷入大麻烦）
3. **限制内的创意解决**：角色用自身缺陷/能力组合解决问题，而非获得新力量
4. **回落到日常+余韵**：问题解决但留有小尾巴（新债务/新麻烦/新关系），给读者追更理由
5. **后日谈/缓冲**：安静章节展示"问题解决后世界是什么样的"

## 大纲一致性检查要点

每次分析时确认：
1. 本章内容与卷纲有无冲突？
2. 本章内容与总纲有无冲突？
3. 角色行为是否符合该阶段的人物弧光？
4. 新埋伏笔是否与已有伏笔表重复？
5. 伏笔回收后是否及时标记？
