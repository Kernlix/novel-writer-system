## 二、新增Domain识别（vs V1-V7基线）

### 🔴 Domain 1: 多角感情线并行/三角关系管理（Multi-Route Romance / Love Triangle Management）

**判断**: ⚠️ **完全新domain，`romance-writer`不覆盖此领域**

V5-V7的感情线是**一对CP的渐进式升温**（和真×惠惠的「握手→被褥→告白→行动确认」四阶段），而V8首次出现**多条感情线在单卷内并行推进**——惠惠的烟花约会、达克尼斯的主动亲吻、和真的「后宫认知→选择焦虑」、克里斯作为旁观者的追问。这不是「恋爱渐进」而是「多角关系的张力管理」。`romance-writer` Agent 专精一对CP的渐进写作，不处理多方感情线的并行调度。

**本能证据（≥8条，远超阈值）：**

| 来源 | ID | 核心 |
|:---|:---|:-----|
| V8ch1 | 第3节 | 惠惠「我果然还是，喜欢这样的你」— 告白级发言（被打断） |
| V8ch3 | sexual-04 | 和真向惠惠邀约看烟花 |

**≥3条**: ✅ **≥11条本能，远超招募阈值**

**建议**: 
- ❌ **不建新Agent**——多角感情线是 `romance-writer` 领域内的进阶技法，非独立写手领域
- ✅ **新增Skill**: `love-triangle-romance`（多角感情线并行/三角关系管理）
  - **归属**: `romance-writer` Agent 的进阶技法扩展
  - **核心技法**:
    1. **并行感情线调度** — 多条感情线索在单卷内同步推进、交错、避免书写死胡同
    2. **性本能爆发与喜剧打断的节奏平衡** — 亲密阈值→喜剧打断→继续推进的三段循环
    3. **三角关系的张力构建** — 「被告知→被追问→被迫选择」的三阶焦虑升温（和真模式）
    4. **旁观者催化剂** — 第三方（克里斯型）追问「你选谁」将隐性多角关系显性化
    5. **「后宫认知」反讽** — 主角以为自己「后宫路线顺利」→ 实际上陷入选择困境的反转
    6. **跨卷钩子技法** — 感情线在卷末故意悬置而非解决（惠惠「今晚来我房间」→第9卷）
  - **触发条件**: 章节涉及≥2个角色对同一主角有恋爱/亲密意图，需调度多条感情线的同步展开与冲突

### 🔴 Domain 2: 神器/无机物拟人化角色写作（Anthropomorphic Object as Full Character）

  - **核心技法**:
    1. **欲望赋予** — 无机物拥有超越设定需求的个人欲望（性偏好、归属渴望、支配欲）

### 🟡 Domain 3: 性本能×喜剧的精密平衡（Erotic-Comedy Balance / Sexual-Instinct Interruption Technique）

**判断**: ⚠️ **边界domain——有独特性，但与Domain 1高度关联**

V8第4章的分析师明确指出「性本能对日常秩序的全面冲击」，并总结了素晴独特的亲密场景处理模式：亲密/紧张感积累→即将突破阈值→喜剧粗暴打断→情感感受留下但喜剧消解尴尬。这是素晴的招牌技法——与一般恋爱喜剧的「暧昧误会导致笑料不同」，素晴是**故意将感情线推到告白/亲吻级，然后用完全无关的突发事件打断**。V1-V7中偶有出现（V5的被褥场景），但V8将其系统化至技法高度。

**≥3条**: ✅ **≥5条本能**

**建议**: 
- ✅ **不再独立新增Skill**——该技法作为 `love-triangle-romance` Skill 的一个核心子模块「喜剧打断的亲密节奏管理」

## 三、最终研判：需新增Agent/Skill

### 核心结论：不需新增Agent，需新增2个Skill

| 优先级 | Domain | 本能数 | 处理方式 |
|:---:|:---|:---:|:---|
| 🔴 **P0** | 多角感情线并行/三角关系管理 | **11+** | **✅ 新增Skill: `love-triangle-romance`** (归属 `romance-writer`) |
| 🔴 **P0** | 神器/无机物拟人化角色写作 | **13+** | **✅ 新增Skill: `anthropomorphic-object-character`** (归属 `humor-writer` 或独立) |
| 🟡 **P1** | 性本能×喜剧打断平衡术 | 5+ | 吸收进 `love-triangle-romance` Skill 作为子模块 |
| 🟡 **P1** | 选美群体兽化叙事 | 4+ | 吸收进 `humor-writer` + `scene-construction` |
| 🟢 **P2** | 女神日常化/神圣解构 | 4 | 已有 `isekai-culture-clash` 覆盖 |
| 🟢 **P2** | 巴尼尔表演艺术 | 3 | 已有 `demon-contract-reversal` 覆盖 |

### 为什么不需要新增Agent？

1. **多角感情线并行**是 `romance-writer` 领域内的进阶技法——`romance-writer` 处理「一对CP的渐进升温」，而新增的 `love-triangle-romance` Skill 处理「多条感情线的并行管理」。两者是从属关系而非并列关系。
2. **神器拟人化角色**虽技术独特但场景集中——阿吉斯是唯一的案例，不足以支撑一个独立Agent。作为Skill挂靠在 `humor-writer` 下（因为阿吉斯本质上是一个喜剧角色）最合理。
3. 已有4个Agent已覆盖素晴所有主要写作领域（喜剧/恋爱/身份悬疑/疑似家族）。新增2个Skill填补技法空白，与V7的处理模式一致。
