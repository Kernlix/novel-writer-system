## 二、新增Domain识别（vs V1-V6基线）

### 🔴 Domain 1: 达克尼斯受虐牺牲型角色心理（Masochistic-Sacrificial Character Psychology）

**判断**: ⚠️ **完全新domain，现有体系无覆盖**

V7将达克尼斯的「受虐-牺牲」本能推至前所未有的叙事核心高度。这**超越**了 `defect-comedy-engine` 中「打不中+抖M」的喜剧标签层面——V7揭示的是：
- 受虐作为**角色存在的底层心理机制**（不只是搞笑）
- 自我牺牲作为**关系绑定/赎罪的方式**
- 痛感-快感联结作为**核心叙事驱动力**（而不只是单场景笑点）
- 「被拥有=被需要」的价值认同扭曲等式

**本能证据**:

| 来源 | ID/内容 | 核心 |
|:---|:--------|:----|
| v7ch2 | `sex-03` 达克尼斯被海德拉压制时的M属性爆发 | 战斗中的性倒错冲动 |
| v7ch2 | `dom-03` 和真命令→达克尼斯服从中获得满足 | 支配-臣服戏剧 |
| v7ch2 | `fight-02` 连续多日单独挑战海德拉遍体鳞伤 | 自毁式赎罪 |
| v7ch5 | `masochistic-desire` 抢婚后"被买下"的性兴奋反应 | 受虐本能巅峰 |
| v7尾声 | 「自我肯定感饥渴本能」完整定义 | 角色心理学核心 |
| v7尾声 | 「一笔勾销」后脸色阴沉的瞬间 | 被拥有=被需要 |
| v7特典 | 读心帽子暴露达克尼斯激烈否认 | 深层害羞反转 |
| v7ch3 | `i4-01` 和真对达克尼斯的惩罚幻想 | 支配与赎罪交叉 |

**≥3条**: ✅ **≥8条本能，远超招募阈值**

**建议**: 
- ❌ **不建新Agent**——「受虐角色写手」过于狭窄，不符合现有Agent命名体系
- ✅ **新增Skill**: `masochistic-sacrificial-character`（受虐牺牲型角色心理写作）
  - 归属: `humor-writer` Agent 或独立
  - 核心技法: 
    1. 自我肯定感饥渴驱动机制（被拥有=被需要）
    2. 痛感-快感联结的叙事化使用（非单纯笑点）
    3. 自毁式赎罪弧线（默默承受→濒临崩溃→被拉回）
    4. 扭曲角色与读者的情感距离管理

## 三、最终研判：需新增Agent/Skill

### 核心结论：不需新增Agent，需新增2个Skill

| 优先级 | Domain | 本能数 | 处理方式 |
|:---:|:---|:---:|:---|
| 🔴 **P0** | 达克尼斯受虐牺牲型角色心理 | **≥8** | **✅ 新增Skill: `masochistic-sacrificial-character`** |
| 🔴 **P0** | 恶魔契约反转叙事 | **≥7** | **✅ 新增Skill: `demon-contract-reversal`** |
| 🟡 **P1** | 社区护卫/集体归属叙事 | ≥7 | 扩展 `scene-construction` |
| 🟢 **P2** | 暴发户/横财心理学 | 5 | 吸收进 `humor-writer` |
| 🟢 **P2** | 育雏本能/善意暴走等 | 7 | 吸收进 `defect-comedy-engine` |

### 为什么不需要新增Agent？

1. **达克尼斯受虐牺牲型角色心理**是「深度角色写作」技法而非独立写手领域——它应该是 `humor-writer` 下的一个高级心理学Skill，不单独成Agent。
2. **恶魔契约反转叙事**是「反派/冲突结构」技法，本质是叙事结构技巧——可吸收进 `plot-outline` 或 `humor-writer`。
3. 已有4个Agent（humor/romance/identity-suspense/pseudo-family）已覆盖素晴所有主要写作领域。新增Skill足以填补空白。

#### 1. Skill: `masochistic-sacrificial-character`（受虐牺牲型角色心理写作）

| 维度 | 说明 |

| **归属** | `humor-writer` Agent 的深度技法扩展 |

| **触发条件** | 创作者需要写「以受虐为心理核心」或「以自我牺牲为赎罪方式」的角色弧线 |
