## 二、领地评分体系——分阶制

当前评分0-100仅适用于**聚落级**。聚落升级后会切换到新的评分框架——不是"破上限"，而是"系统进化"（符合封印管理终端的设定）。

| 阶位 | 对应等级 | 评分范围 | 覆盖卷 | 解锁条件 | 新增评估维度 |
|:--:|:--|:--:|:--:|:--|:--|
| **聚落级** | 村庄 | 0-100 | Vol 1-3 | 初始 | 人口、粮食、防御、基础经济 |
| **城镇级** | 镇子 | 0-200 | Vol 4-6 | 人口800+、城墙、稳定贸易路线 | **军事、外交、种族多样性、工业** |
| **城邦级** | 城市 | 0-500 | Vol 7-8 | 人口3000+、多元种族、正式军队 | **政治影响力、魔法基建、战争产能** |
| **领地级** | 区域首府 | 0-1000 | Vol 9 | 终局阶段 | **封印网络覆盖、位面影响力** |

**评分换算方式（不以数字直接继承）：**
- 聚落级100分 → 解锁城镇级后，按比例折算为**城镇级**的起始分（约30-40分）
- 主角面板看到的不是"分数涨了"，而是**系统界面本身变了**——新的评估维度出现，旧的维度权重下调
- 例如：聚落级最看重"粮食自给"，城镇级最看重"经济活力和军事投送"

**这样设计解决了：**
1. 不会出现"Vol 2就62分，Vol 9怎么办"的问题——因为Vol 3升阶后评分体系重置了
2. 每次升阶给读者新鲜感——新的面板、新的维度、新的挑战
3. 符合系统设定——封印管理终端会随着领地规模的扩大自动调整评估算法

## 三、人口增长——有来源、有瓶颈、有消耗

| 卷 | 人口 | 增长来源 | 说明 |
|:--:|:--:|:--|:--|
| Vol 1 | 187→280 | 流民收容+降兵转化 | 主动政策 |
| Vol 2 | 280→400 | 边境流民+商队定居+新生儿 | 领地名气外传，开始有人主动来投 |
| Vol 3 | 400→550 | 子爵领难民+矮人移民+自然增长 | 子爵领瓦解后人口向西流动 |
| Vol 4 | 550→800 | 城镇升级吸引移民+种族多样化 | 矮人工坊吸引矮人家属，精灵幸存者 |
| Vol 5 | 800→1300 | 贸易枢纽效应+兽人部落融合 | [[狼牙]]的部族残余汇入 |
| Vol 6 | 1300→2500 | 战乱难民潮 | 深渊威胁扩散，周边人口涌入避难 |
| Vol 7 | 2500→5000 | 大规模逃难 | 全面战争，主角领地成为安全区 |
| Vol 8 | 5000→8000 | 战争+瘟疫消耗与补充并存 | 战损与流入并行 |
| Vol 9 | 8000→12000 | 战后重建 | 新秩序稳定，人口恢复 |

**人口限制条件（避免无脑增长）：**
- **粮食瓶颈**：耕地有限，扩产需要时间。Vol 2-3的粮食储备只能支撑3个月
- **住房瓶颈**：每次人口暴增后需要至少几章搞建设（种田文的核心爽点）
- **疾病风险**：人口密度高了会爆发瘟疫（可做支线剧情）
- **战争损耗**：Vol 7-8会有大量战损，人口增长≠只增不减

| 1 | 巴尔+鸦+米娅 | 流浪冒险小队 | 专业战斗+情报侦察+魔法 | 领地从民兵自卫到专业武力的跨越 |
| 1 | 铁锤·铜须 | 铁炉堡（出走） | 矮人锻造术、黯钢技术 | **领地经济的支柱，黯钢体系奠基人** |

| 3 | 矮人学徒1-2名 | 铁炉堡商队 | 基础锻造+冶炼 | 锻造体系从单人→团队的转折 |
| 3 | 游方郎中 | 难民 | 基础医术 | 领地第一个正式的医疗保障 |

| 4 | 精灵幸存者小队 | 月影森林方向 | 精灵战斗术+附魔知识 | 种族多样化+魔法技术加成 |
| 5 | 兽人部落残余 | 东部荒原（狼牙线） | 更多训狼师、部落战士 | 狼骑兵大规模扩编 |

> 关键原则：
> 1. **每个新角色都改变领地的能力边界**——不是凑人数，是解锁新功能
> 2. **人才来源多样化**——有主动投奔的、有任务招募的、有名声吸引的、有战争难民的
> 3. **早期人才是主角"捡"来的，中期是"招"来的，后期是"投奔"来的**——反映领地地位上升

## 五、主角变强三维体系

如果**变强=升级=腐化**，那主角到后期只有两个选择：①停在小等级不升级→太弱打不过 ②升级→被腐化→崩了。

**解决方案：主角的力量来源从一条腿变成三条腿。**

```
┌──────────────────────────────────────────────┐
│              主角战力体系                       │
├─────────────┬──────────────┬──────────────────┤
│   ① 等级反馈  │  ② 深渊之力    │  ③ 封印技术       │
│  (系统白道)   │  (高风险高回报) │  (控制/可持续)     │
├─────────────┼──────────────┼──────────────────┤
│ 等级提升→    │ 腐化值增长→    │ 研究封印技术→     │
│ 基础属性↑    │ 深渊抗性↑      │ 内封印回路→       │
│ 解锁系统功能  │ 获得深渊能力   │ 抑制腐化+反制手段  │
│ 代价：腐化↑   │ 危险：失控↑   │ 代价：研发时间↑    │
└──────────────────────────────────────────────┘
```

### 路径①：等级反馈（Vol 1-3 主力，之后辅助）
- 每升一级：力量、速度、感知、恢复基础提升
- Vol 4之后不再是主要手段（腐化代价太高）
- Vol 1-2：黑铁→青铜；Vol 3：青铜→白银；之后升级节奏大幅放缓

### 路径②：深渊之力（Vol 3+解锁，伴随腐化值解锁能力）

| 腐化值 | 解锁能力 | 揭示时机 |
| :--: | :-- | :-- |
| 10%+ | 深渊感知（能感应裂隙和深渊生物位置） | Vol 3初：主角先发现"莫名其妙能感知到矿井深处的异常"——以为是直觉，后来才确认是腐化值带来的能力 |
| 20%+ | 临时深渊强化（短时间boost战力，使用后虚弱） | Vol 3中：第一次无意识触发——读者先看到效果，主角几章后才确认原因 |
| 35%+ | 威慑/统率（低阶深渊生物视主角为"同类"） | Vol 4末：在深渊裂隙深处遭遇大量低阶深渊生物时，首次发现它们不攻击自己——[[莉莉丝·暗影]]解释"它们认为你是同类" |
| 50%+ | 半深渊化战斗形态（主动激活，外观改变，战力大幅提升） | Vol 6高潮：第一次主动激活——本卷最大的"哇塞"时刻 |
| 70%+ | 深渊意志干涉（极高风险，与深渊争夺控制权） | Vol 8：终局前的最强也是最危险的形态 |

### 路径③：封印技术（Vol 4 首次突破，后期核心）
- Vol 4：建立体内封印回路 —— **腐化增长速度从"每级1-2%"降至"每级0.3-0.5%"**
- Vol 5-6：封印成熟——可以主动释放"封印技"：黯钢凝结、符文附魔、能量中和
- Vol 7-8：封印+深渊融合技——用封印技术驾驭深渊之力，形成独特战斗风格
- Vol 9：最终突破——完全掌握自身能量循环

### 三维体系的相互作用

```
等级↔深渊：等级越高，深渊之力越强，但腐化越快 → 需要封印来平衡
深渊↔封印：封印抑制腐化，但也限制深渊之力输出 → 需要找到平衡点
封印↔等级：封印技术越成熟，升级的腐化代价越低 → 可以安全升级
```

**主角的核心成长弧不是"升级变强"，而是在三条路之间寻找平衡——这是贯穿全书的人物内核。**

## 九、矮人氏族体系

山地矮人以铁炉堡（铁炉山脉）为城邦中心，氏族制度是其社会核心。四大氏族各司其职：

| 氏族 | 专精领域 | 定位 | 剧情关联 |
|:--|:--|:--|:--|
| **铜须氏族** | 锻造·精品路线 | 铁炉堡最古老的锻造氏族，掌握核心熔炼技术，每一件装备精工细作 | [[铁锤·铜须]]所属，主线角色 |
| **钢鬃氏族** | 军备·批量路线 | 主张市场化量产，向外销售获取影响力，与铜须存在路线之争 | 保守派主力，铁锤出走原因 |
| **石炉氏族** | 采矿与地质勘探 | 中立派，只关心矿脉，不参与政治，通过商队传递信息 | 未来深渊异常的报告者 |
| **铁卫氏族** | 城防与军事训练 | 效忠铁炉堡整体而非某一氏族，在冲突中充当调停者 | 未来矮人援军来源 |

**路线之争（铁锤出走的真正原因）：** 铜须坚持传统精工，钢鬃主张批量市场化。铁锤之父（老族长）倾向于传统，但钢鬃氏族联合保守派持续施压。铁锤不是因叛逆出走——他是替父亲扛下冲突，主动离开以减少族内分裂。

## 十、兽人部落体系

兽人没有共居城邦，荒原广袤，各部落**各自为政**，语言和文化已有分化。共同的处境是被人类王国视为"蛮族"，不断被压迫驱赶。

| **血蹄部落** | 荒原角兽（巨角野牛） | 血蹄荒原·中部 | 重装冲锋，正面碾压 | 崇尚力量，部落议会制，人口最多 | 被压迫但未溃散 |
| **裂风部落** | 荒原猎鹰 | 东部丘陵 | 轻装游击，侦查标枪 | 崇尚敏捷与智慧，擅长追踪 | 血蹄的长期盟友 |
| **赤牙部落** | 獠牙野猪 | 荒原南缘 | 狂暴冲锋，悍不畏死 | 最激进好战，最不被人类接纳 | 被重点打压，近溃散 |

**与黑狼部落的关系：** 黑狼盘踞北境，与东部三大部落隔着拜伦王国领土，没有直接往来。东部部落没有驯狼传统——血蹄乘骑角兽或步战，裂风轻步兵游击，赤牙狂暴步战。驯狼是北境黑狼独有的传统。

## 十一、半人马种族体系

半人马是帝国后期的多元种族之一。帝国发展需要马匹（军事骑兵、运输、农耕通讯），但若走普通养马路线则与普通领主种田无异——半人马自身就是骑兵，引入后与训狼师形成互补的骑兵体系。

### 栖息地：苍风草原

大陆西陲的广袤草原，位于拜伦王国以西，中间隔着三层天然屏障：

| 层次 | 名称 | 宽度 | 特征 |
|:--|:--|:--:|:--|
| 第一层 | 东部丘陵/疏林地带 | ~80里 | 人类活动边界到此为止 |
| 第二层 | **龙骨山脉** | ~100里 | 南北走向，高耸险峻，仅有几个山口，冬季大雪封山 |
| 第三层 | **幽暗森林**（+大沼泽） | ~150里 | 山脉西麓，终年潮湿多雾，魔兽出没，旧帝国哨站早已废弃 |
| **合计** | 屏障总宽 | **~330里** | 单向穿越需7-10天（有向导），人类未深入 |

穿越屏障后的苍风草原：
- 东西纵深 **~700里**，南北跨度 **~1200里**，草原中部水草最丰美
- 气候干冷，冬季多暴风雪，**不适合农耕**但**适合放牧**
- 无重要矿产——人类王国从未西扩的根本原因
- 以黑石镇为原点，到苍风草原核心区约 **800里**（含屏障穿越），有向导需2-3周

### 种族基础

| 项目 | 内容 |
|:--|:--|
| 社会形态 | 游牧氏族制，逐水草而居，无固定城邦 |
| 寿命 | 约120-150年 |
| 外貌 | 人躯高1.8-2.0m，全高2.4-2.7m，古铜色皮肤，马躯为草原耐力型 |
| 语言 | 半人马语（氏族内使用），部分通过商队接触通晓通用语 |
| 最高权力 | **氏族议会**（战时推举战争酋长） |

### 五大氏族

| 氏族 | 职责 | 文化特征 |
|:--|:--|:--|
| **逐风氏族** | 先锋·侦察 | 体型最轻捷，弓术最精，负责探路和外围警戒 |
| **铁蹄氏族** | 战士·冲锋 | 体型最大，近战主力，长矛集群冲锋 |
| **苍牧氏族** | 牧马·育养 | 掌握与马群的天赋沟通能力，负责管理全族马群 |
| **星语氏族** | 萨满·先知 | 祭祀、医疗、观星占卜，保存族群历史与传说 |
| **灰鬃氏族** | 长老·议会 | 全族最年长者组成，跨氏族议事，裁决纠纷 |

### 与马的关系

**不骑马（实际原因，非禁忌）：** 半人马自己跑得够快，骑在马上反而两条躯干互相妨碍，单纯是没必要。

**不吃马肉（文化禁忌）：** 视马为"未开化的远亲"，吃马肉等同于同类相食，违者被氏族永久驱逐。

**牧马共生天赋：** 半人马与野生马群建立共生关系而非暴力驯服。在马群附近扎营，通过长时间"共存"让马群习惯他们。苍牧氏族和星语氏族成员能通过**情绪感知**与马群头马建立联系——一旦头马信任半人马，整个马群都会跟随。半人马管理的马群不需要缰绳、马鞍、鞭子。

**护蹄体系（非马蹄铁）：** 草原半人马每天跑几十里，硬质草甸和沙砾地是天然磨蹄石，蹄壁生长与磨损自动平衡。灰鬃/苍牧氏族有人用特制骨制/石制蹄刀定期修整蹄缘。极端地形用生革蹄套（绑式，非钉入）。人类钉蹄铁是因负载重+走石板路磨损不均，半人马不面临此问题。

### 文化冲突点

半人马首次接触人类领地时可能爆发摩擦，保留以下冲突点：

| 冲突点 | 半人马感受 |
|:--|:--|
| 人类用鞭子抽马 | 反感——他们用共存方式牧马，暴力不可接受 |
| 人类吃马肉 | 最严重冲突，可能导致流血事件 |

**解决路径：** 主角做出调整——领地马匹管理逐步移交半人马，帝国内部禁止马肉（补给线仍可用但远离半人马聚居区）。

### 军事定位

与训狼师互补而非替代：

| 兵种 | 速度 | 地形 | 战术定位 |
|:--|:--|:--|:--|
| 训狼师 | 高 | 平原+山地+森林 | 近战冲锋，威慑力强 |
| 半人马弓骑 | 极高 | 开阔平原 | 远程游击，风筝战术 |
| 半人马铁蹄冲锋 | 高 | 开阔平原 | 重装穿刺破阵 |
| 人类骑兵 | 中 | 各种地形 | 常规冲锋+后勤 |

半人马战士自身即是骑兵，帝国人类骑兵照样存在——人类骑兵使用半人马饲养的战马。

### 深渊暗线

星语氏族萨满在观星中感知到"东方的深渊气息在蔓延"，且在西海岸发现了深渊污染的迹象——深渊不仅从地下渗漏，也可能从世界尽头的海面之下蔓延。半人马东迁的深层原因与此有关。

## 十二、宿主传承与深渊转化发现

> 封印管理终端的历代宿主编年史，以及深渊能量转化理论的发现与传承。

### 宿主编年史

| 代次 | 时期 | 身份 | 结局 | 备注 |
|:--:|:--|:--|:--|:--|
| 初代 | ~5000年前 | 旧帝国封印师 | 失控 | 建立封印网络时被深渊能量反噬 |
| 二代 | ~3000年前 | 旧帝国后裔 | 失忆（任务完成被抹除记忆） | 成功完成封印加固 |
| 三代 | ~800年前 | 罗塞尔公国大法师 | 解除绑定，部分失忆 | **发现深渊能量转化理论**，留下手稿 |
| 四代 | ~200年前 | [[莉莉丝·暗影]] | 解除绑定，保留知识 | 幸存者，目前活跃 |
| 五代 | **当代** | [[艾伦·默]]（主角） | 进行中 | 因封印提前松动被紧急召唤 |

### 三代宿主与转化发现

约800年前，一名罗塞尔大法师被系统选中为宿主。他发现了颠覆性的真相：
- 腐化值并非纯粹的"诅咒"——它是一种可被**重新编程**的能量形态
- 深渊能量进入体内产生腐化值，本质上是能量格式从无序到可度量的转换
- 高腐化值不只是风险，也是一种"使用权"解锁（即双刃剑机制）
- 理论上，构建精密能量回路可以将腐化值"固化"为永久能力

发现记录为《深渊能量转化七论》，手稿完整留在了罗塞尔法师议会。

### 罗塞尔法师议会的知识传承

**知道**：深渊能量可被转化利用；封印技术的理论基础是转化理论的分支；封印存在周期性老化特征。

**不知道**：系统是封印管理终端；沉睡之神存在；封印网络全貌和中央枢纽位置；宿主机制全部细节。

**对主角的意义**：Vol 4+主角接触罗塞尔时，对方发现主角的"封印技术"思路与他们千年研究高度吻合→深度合作。罗塞尔提供转化理论，主角提供系统数据和实战验证。罗塞尔的知识是Vol 7+主角实现"转化/固化"的关键技术前提之一。

## 十三、宿主魂穿机制

> **创作参考/暗线规划**——穿越机制的底层原理属于终局揭示（Vol 8-9），正文前期不需要直接交代。但设计者需理解这套机制以保证剧情逻辑的一致性。

### 核心矛盾：为什么不在本地找宿主？

这个世界的**所有灵魂从诞生起就带有深渊能量的"底色"**——因为世界本身就是封印场，深渊能量无处不在。系统如果绑定本地灵魂，相当于在对方的灵魂上运行一个程序，但这个操作系统早就被深渊"感染"了。本地灵魂会被深渊反向影响、策反甚至预判。

**解决方式**：系统在初代宿主失控后（旧帝国封印师，本地灵魂绑定），修改了核心协议——转而从**不存在深渊的世界**召唤灵魂。跨世界灵魂的"信息底版"是干净的，深渊无法预判、无法标记、无法追踪。

### 魂穿的底层逻辑：不是主动抓捕，是广播匹配

系统不做"跨世界抓人"——它没那个能力。系统做的是一套**信标广播+自然共振**的筛选机制：

```text
┌───────────────┐         ┌──────────────────┐         ┌───────────────┐
│  系统（此世界） │ ──①──→ │  世界屏障薄弱处   │ ──②──→ │  地球（无魔力）│
│  发出共振信标  │         │  信息渗透/过滤    │         │  灵魂扫描匹配  │
└───────────────┘         └──────────────────┘         └───────────────┘
                               │ ③匹配成功
                               ↓
                         ┌──────────────────┐
                         │  灵魂自然穿越      │
                         │  （非系统拉动，     │
                         │   是灵魂被吸引过来） │
                         └──────────────────┘
```

**七步流程**：

| # | 阶段 | 发生什么 | 备注 |
| :-: | :-- | :-- | :-- |
| ① | **检测** | 系统检测到封印老化超过阈值→判定需要新宿主 | 提前数年触发，不是临时决定 |
| ② | **调频** | 系统聚焦于已记录的兼容世界（以地球为主），扫描灵魂频率 | 5000年来持续优化筛选算法 |
| ③ | **放标** | 系统在世界屏障薄弱处打开微裂隙，发出共振信标 | 裂隙极小，仅允许信息而非能量通过 |
| ④ | **共振** | 目标灵魂的灵魂频率与信标匹配→灵魂"听到"了召唤 | 表现为莫名其妙的既视感、执念、梦境 |
| ⑤ | **穿越** | 灵魂脱离原身体，沿屏障裂隙"漂流"过来 | **记忆在此过程中被大量剥离**——屏障自带信息过滤 |
| ⑥ | **着床** | 灵魂被导向一具匹配的身体：刚死亡、身体完整、原灵魂已离开 | 身体需经过预先"调校"（系统在放标前已锁定目标） |
| ⑦ | **绑定** | 系统与灵魂建立连接，完成初始化 | 灵魂进入新身体后有一段时间的混沌期 |

### 为什么是地球？

不是"随机选了一个世界"，而是**经过5000年优化筛选出的最优解**：

1. **信息隔离最彻底**：地球完全没有超自然能量，灵魂从未接触过任何魔力/深渊波动——"最干净的底版"
2. **意识结构兼容**：地球人类能理解抽象信息系统（数据面板、状态栏、任务提示），如果是太原始的世界，宿主可能根本看不懂系统界面
3. **程序员偏好**：经过多代测试，系统发现"具有逻辑系统化思维"的灵魂与终端管理模式的配合效率最高——这不是巧合，是选择压

### 陈默的特殊性（与终局关联）

**表层"合理"解释**：频率匹配上的幸运儿，恰好有程序员思维，恰好被选中。

**Vol 8-9揭示的真相**：他的穿越不是纯随机匹配——

旧帝国的初代封印师团队在设计终端时，预见到了封印网络终有老化的一天。他们在终端的核心协议中嵌入了**紧急召唤协议**：将一部分自己的灵魂信息编码为"种子协议"。当封印网络濒临崩溃时，终端会自动激活协议——不是召唤某个特定的人，而是**改变放标频率，使只有携带特定灵魂特征的灵魂才能共振**。

陈默的灵魂携带了与初代封印师**同源的结构印记**——他不需要是后代或转世，而是协议本身就像一把"锁"，只有特定"钥匙结构"的灵魂才能匹配。

**这解释了一切**：

- 为什么他对深渊有天然亲和力——终端设计的宿主品质要求
- 为什么他与系统配合得天衣无缝——他是被"预设"为适配系统的灵魂
- 为什么他不是"纯外来者"——他的灵魂结构来自这个世界的初代封印师团队
- 为什么他能在终局承受深渊核心——他是被设计成"钥匙"的

### 穿越代价与限制

| 代价/限制 | 说明 |
| :-- | :-- |
| **记忆剥离** | 穿越时大部分记忆丢失。陈默保留较多——不是正常的，是协议匹配度高的附带效果 |
| **单向** | 无法返回原世界。屏障裂隙只允许单向信息流 |
| **能量成本高** | 每次宿主召唤消耗终端大量储备能量，间隔至少以百年为单位 |
| **不可重复选同一人** | 一个灵魂只能被召唤一次 |
| **屏障损伤** | 每次穿越在世界屏障上留下微裂隙——裂隙会自行愈合，但多次召唤会累积薄弱点 |

### 与其他角色/设定的一致性

- **[[莉莉丝·暗影]]不是魂穿宿主**：她（第四代）是本地灵魂直接绑定，是系统在封印加速松动时的"紧急手动绑定"。这解释了为什么她能解绑——本地球生灵魂有"根"，挣脱代价虽然大但可能；跨世界灵魂没有根，一旦绑定几乎无法主动脱离
- **三代（罗塞尔法师）任务结束后被抹除部分记忆**：这是正常的"退出机制"——系统在宿主完成任务后剥离穿越相关的记忆，防止知识泄露
- **初代（旧帝国封印师）失控**：因为是本世界灵魂直接绑定，没有跨世界过滤层的保护，灵魂与深渊的交互更直接、更危险
- **主角解绑的可能性**：比莉莉丝更难——他没有"本地锚点"，但如果他能在这个世界建立起足够强的"锚"（人际关系、领地责任、自我认同），理论上可以尝试。这也是Vol 9终局选择的一种路径
