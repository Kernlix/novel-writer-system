# ⏱️ 弹药倒计时张力 Skill (Ammo Countdown Tension)

> **核心来源**：《十日终焉》第496-497章——宋七的爆燃弹药（碎片）逐渐耗尽，信念波动导致攻击失败，读者能精确感受到他的生存窗口在缩小
> **提炼方法**：从生死张力、策略与博弈维度中提取的紧迫感设计技法
> **适用场景**：消耗战、资源受限的战斗、需要持续紧迫感的场景
> **与 `fear-curve-power-display` 的区别**：fear-curve-power-display 通过他人恐惧展示力量；本Skill 通过**资源可视化创造紧迫感**——不展示敌人的强大，而是展示我方资源的枯竭

---

## 核心原理

**弹药倒计时**的核心是：将角色的资源（弹药、信念、体力等）可视化为「倒计时」——读者可以精确计算角色还剩多少资源、还能坚持多久，这种「可量化的绝望」比模糊的威胁更有压迫感。

### 核心公式

```
普通张力：敌人很强大 → 读者感到紧张（模糊）
弹药倒计时：角色还剩X次机会 → 读者精确计算生存概率 → 读者感到焦虑（精确）
```

### 关键区别

| 模糊威胁型张力 | 弹药倒计时型张力 |
|:---------------|:-----------------|
| 「敌人很强」 | 「还剩3次机会」 |
| 读者感到紧张 | 读者感到焦虑 |
| 张力来源：敌人的强大 | 张力来源：资源的枯竭 |
| 读者被动感受 | 读者主动计算 |

---

## 技法1：资源可视化 (Resource Visualization)

将角色的资源（弹药、信念、体力）以具体、可计数的方式呈现——让读者清晰看到资源的消耗过程。

### 1.1 核心原则

- 资源必须「可计数」——弹药数量、手指数量、信念强度等
- 每次消耗必须「可见」——读者看到资源减少
- 剩余资源必须「可计算」——读者能算出还剩多少
- 资源耗尽的后果必须「可预见」——读者知道耗尽意味着什么

### 1.2 宋七案例分析（496-497章）

**弹药可视化**：
- 宋七的弹药是「碎片」（木屑、碎石、小石子）
- 每次攻击消耗一个碎片
- 读者可以数：他还有几个碎片？

**信念可视化**：
- 爆燃需要信念支撑
- 信念不足时攻击失败（木屑未爆炸）
- 读者可以看到：他的信念在波动

**身体可视化**：
- 497章：左手完全烧焦发黑，手指断裂
- 498章：断裂手指作为投掷武器（五根手指=五次机会）
- 读者可以数：他还剩几根手指？

## 技法2：失败预演 (Failure Preview)

在资源耗尽之前，先展示一次「信念不足导致失败」——让读者看到失败的样子，从而更害怕下一次失败。

### 2.1 核心原则

- 失败必须「可见」——读者清楚看到攻击失败（木屑未爆炸）
- 失败必须「有原因」——信念不足（不是随机失败）
- 失败必须「可重复」——如果信念继续波动，失败会再次发生
- 失败的后果必须「严重」——一次失败可能意味着死亡

### 2.2 宋七案例分析（496章）

**失败展示**：
- 宋七两次抛出木屑，都未引发爆炸
- 原因：信念动摇（「此时若是出现慌乱或是怕死的念头……那死的就真是自己了」）
- 后果：浪费了两次弹药机会，地兔趁机逼近

**读者反应**：
- 「他信念不够了！」→ 读者开始担心下一次攻击
- 「如果再失败怎么办？」→ 焦虑感升级
- 「他还剩几个碎片？」→ 开始倒计时

## 技法3：双重倒计时 (Dual Countdown)

同时运行两个倒计时——资源倒计时（弹药/手指）+ 信念倒计时（心理状态），双重倒计时叠加制造更强的紧迫感。

### 3.1 核心原则

- 两个倒计时必须「独立运行」——资源倒计时是物理的，信念倒计时是心理的
- 两个倒计时必须「相互影响」——资源减少→信念动摇→攻击失败→资源浪费
- 两个倒计时必须「同时耗尽」——资源耗尽+信念崩塌=最终摊牌
- 读者需要同时跟踪两个倒计时——信息密度增加→焦虑感加倍

### 3.2 宋七案例分析

**资源倒计时**：
- 碎片数量逐渐减少
- 手指数量固定（五根=五次机会）
- 每次攻击消耗一个资源

**信念倒计时**：
- 战斗持续→信念波动
- 信念波动→攻击失败→浪费资源
- 资源减少→信念进一步动摇

**双重叠加**：
- 碎片少了→每次攻击更重要→信念压力更大
- 信念波动→攻击失败→碎片浪费→资源更少
- 恶性循环→读者焦虑感持续升级

## 技法4：倒计时归零的释放 (Countdown Zero Release)

当倒计时归零（资源耗尽）时，不是叙事结束，而是角色弧线的终极完成——用最后的资源完成最后一次攻击。

### 4.1 核心原则

- 倒计时归零必须「有仪式感」——不是突然结束，而是角色主动选择
- 最后一次攻击必须「有明确目的」——不是乱打，而是为了特定战术目标
- 归零后的角色状态必须「可预见但不可逆转」——读者知道他会死/重伤，但无法阻止
- 归零的冲击力与倒计时长度成正比——倒计时越长，归零越震撼

### 4.2 宋七案例分析（497章）

**倒计时归零**：
- 碎片耗尽 → 宋七用拳头贴脸爆燃（身体武器化）
- 手指断裂 → 用断裂手指投掷（最后的弹药）
- 信念崩塌 → 「走之前杀了我」（用死亡拖延）

**归零的仪式感**：
- 「谁说没有爆炸物……我就爆炸不了？」（主动选择）
- 代价：左手完全烧焦（不可逆转）
- 效果：夺取法宝+拖延时间（有明确目的）

## 写作要点

| 要素 | 说明 |
|:-----|:-----|
| **可计数性** | 资源必须可计数（碎片数量、手指数量、信念强度） |
| **可见消耗** | 每次消耗必须可见（读者看到资源减少） |
| **失败预演** | 先展示一次失败（信念不足→攻击失败），让读者害怕下一次 |
| **双重倒计时** | 同时运行资源倒计时+信念倒计时，双重叠加 |
| **恶性循环** | 资源减少→信念动摇→攻击失败→资源浪费（恶性循环） |
| **归零仪式** | 倒计时归零必须有仪式感（角色主动选择最后一次攻击） |

## 适用章节类型

- 消耗战（资源有限的战斗）
- 以弱胜强的力量不对等对抗
- 需要持续紧迫感的场景
- 角色弧线的「最后阶段」（从战斗到牺牲）
- 密室/封闭空间中的生存战
