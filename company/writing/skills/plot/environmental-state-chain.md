# 🔥 环境状态叠加 Skill (Environmental State Chain)

> 油球留下油层 → 铁球产生火花 → 引燃油层 → 全员火海——每一击都在为下一击做铺垫

---

## 核心原理

**环境状态叠加**是指：前序攻击在环境中留下的**残留效果**（油层、碎屑、裂缝、冰面等），被后续攻击**激活或放大**，创造出超越单次攻击维度的复合威胁。

**与「三维威胁叠加」的本质区别**：
- **三维威胁叠加**：三种**独立的**物理风险**同时**作用（高速攻击+不稳定地面+潜伏威胁）
- **环境状态叠加**：前序攻击的**残留效果**被后续攻击**链式触发**（油→火花→大火）

**叙事价值**：
- 将环境从「被动背景」升级为「**主动参与者**」——环境本身成为武器
- 创造**复合威胁**——最终伤害 = 攻击A × 攻击B × 环境放大
- 展示**战场累积效应**——不是单次攻击决定胜负，而是所有攻击的残留效果共同决定
- 为**团队协作**提供战术维度——需要有人清除环境状态（灭火），而不仅仅是躲避攻击

**心理机制**：
- **链式反应的恐惧**：一个事件触发另一个事件，失控感
- **累积压力**：每次攻击不仅造成直接伤害，还为下一次攻击做铺垫
- **环境不可控**：角色可以躲避攻击，但无法完全清除环境残留

---

## 技法要素

### 1. 状态残留设计

前序攻击必须在环境中留下**可感知的物理残留**——不是抽象的「伤害」，而是具体的物质变化。

**残留类型**：
- **液体残留**：油层、水迹、毒液——增加滑倒/引燃/腐蚀风险
- **固体残留**：碎屑、碎片、裂缝——影响移动/提供掩体/制造障碍
- **热量残留**：余温、火焰、蒸汽——持续伤害/改变环境温度
- **结构残留**：断裂的栏杆、坍塌的墙壁——改变地形/创造新路径

### 2. 链式触发机制

后续攻击必须能够**激活或放大**前序残留——不是独立作用，而是**协同作用**。

**触发设计**：
- **物理触发**：火花+油层=大火；碎片+冲击波=飞射物
- **化学触发**：毒液+水=扩散；热量+油=蒸气
- **结构触发**：裂缝+重力=坍塌；断裂+推力=飞射

### 3. 环境角色的升级

环境从「被动背景」升级为「**主动参与者**」——它有自己的「行为」（蔓延、扩散、燃烧）。

**升级设计**：
- 环境变化有**自己的时间线**（油层慢慢扩散，大火慢慢蔓延）
- 环境变化**不受角色控制**（一旦引燃，无法停止）
- 环境变化**影响所有角色**（不分敌我，全场波及）

## 适用场景

| 场景类型 | 适用度 | 说明 |
|:---------|:------:|:-----|
| 大规模战斗/群战 | ★★★★★ | 最经典的环境状态叠加场景 |
| 死亡游戏/竞技场 | ★★★★★ | 封闭空间的累积效应 |
| 自然灾害/灾难 | ★★★★☆ | 灾害的链式反应 |
| 追逐/逃亡 | ★★★☆☆ | 逃亡路线上的累积障碍 |
| 单挑/决斗 | ★★☆☆☆ | 环境元素较少 |

## 与其他技法的关系

| 关联技法 | 关系 |
|:---------|:-----|
| 环境状态叠加 vs 三维威胁叠加 | 三维威胁 = 同时存在的独立风险；环境状态叠加 = 前序残留被后续触发——前者是「并行威胁」，后者是「串行放大」 |
| 环境状态叠加 vs 场景即武器 | 场景即武器 = 利用场景的固定属性；环境状态叠加 = 利用场景的动态变化——前者是「静态地形」，后者是「动态战场」 |
| 环境状态叠加 vs 感官减法场景 | 感官减法 = 剥夺感官制造恐惧；环境状态叠加 = 累积状态制造复合威胁——前者是「减少」，后者是「增加」 |

## 实例参考

**十日终焉第469-470章「搏命」：**
- **状态1**：地马投出油球 → 油层覆盖冰面（液体残留）
- **状态2**：乔家劲用球棒挡铁球 → 火花飞溅（热量残留）
- **状态3**：火花接触油层 → 大火蔓延（链式触发）
- **状态4**：大火从乔家劲手臂蔓延到全身 → 到冰面 → 到所有人皮衣（环境角色升级）
- **复合伤害**：最终伤害 ≠ 铁球撞击 + 油层 + 火灾，而是 **铁球 × 油层 × 火灾 × 冰面（滑倒）× 皮衣（助燃）**

**灭火的战术价值**：
- 罗十一的「忘忧」消除痛感 → 乔家劲不知轻重地扑火
- 齐夏的泥球（沉默预备动作）→ 涂抹灭火
- 皮衣覆盖 → 隔绝氧气
- 三种灭火方式对应三个角色——灭火本身也是团队协作

**设计精妙之处**：
- 油球不是「攻击」——它是「铺垫」。地马的真正攻击是铁球，油球只是为火灾做准备
- 火灾不是「意外」——它是地马的**战术设计**。油+铁+球棒=预谋的火攻
- 环境状态叠加创造了**双重紧迫**：躲避攻击（即时威胁）+ 清除环境状态（持续威胁）
