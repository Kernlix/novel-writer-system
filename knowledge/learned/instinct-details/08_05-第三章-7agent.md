# 七维本能分析 — 素晴第8卷第3章「将一切交给这个能干的顾问！」

基于七维本能分析框架（7つの本能）对本章角色行为与叙事驱动力进行拆解。

---

## 本能冲突与共振

| 冲突/共振 | 说明 |
|-----------|------|
| 性本能 vs 支配本能 | 和真的「看板娘穿泳装」提案本质上是性本能的表达，但他是通过支配本能（顾问身份的决策权）来实现的——性和支配形成高效联动：「有权就能实现欲望」是这个角色的核心引擎 |
| 闘争本能 vs 生存本能 | 惠惠在森林中强行使用爆裂魔法（闘争本能压倒生存判断）引发了数百只虫群的追杀，导致全员撤退——闘争的快感以生存危机的形式反噬集体 |
| 群居本能 × 収集本能 | 阿库西斯教团全体参与的摊位运营——群居本能让人团结，但収集本能让团结变质为"集体骗钱"——蝌蚪、假龙、乌贼烧，社群的温暖变成商业欺诈的帮凶 |
| 性本能 × 群居本能 | 魅魔店员在庆功宴上的出现——和真的性本能驱使他留在酒席，但宴会本身是群居仪式（全体委员举杯），性本能和群居本能在此高度共振，让和真放弃了原本的辞职决定 |
| 支配本能 vs 群居本能 | 和真在委员会中独断专行（支配本能）与委员们发生冲突，但最终目标（祭典成功）又需要集体协作（群居本能）——支配必须妥协于群居才能实现成果 |
| 探求本能 × 支配本能 | 和真对阿库娅反常行为的持续探究（探求），却始终没有追问到底——为了维持他对阿库娅"笨蛋角色"的支配性认知框架，他选择性地拒绝了深入探求，防止自己的世界观被颠覆 |
| 生存本能 vs 闘争本能（达克尼斯） | 蚂蚁爬满全身、铠甲无法脱下——身体处于极度不适状态（生存本能报警），但达克尼斯的反应是「真是新鲜的感觉」（性本能/闘争本能的混合）——三种本能在同一刺激下的分裂应对 |
| 収集本能 vs 群居本能（阿库娅） | 阿库娅自掏腰包让教团参与祭典——収集本能（囤积金钱）和群居本能（让教团被认可）的冲突，最终群居本能胜利，却导致了教团大规模亏损的荒诞结局 |

## 章节本能画像

本章是典型的素晴"祭典系"章节——7种本能在祭典这个特殊场景下同时活跃：

- **和真**：性本能（欲望动机）× 支配本能（执行手段）× 収集本能（最终目标）形成闭环，成为全章最完整的本能驱动链条
- **阿库娅**：群居本能（想被认可）× 生存本能（反常的克制）之间张力最大，但収集本能（自掏腰包亏本）让她一如既往地"努力却失败"
- **达克尼斯**：闘争本能（迎击最多敌人）和性本能（M属性快感）在本章深度融合
- **惠惠**：闘争本能（非要放爆裂魔法）压倒所有理性计算，但她的烟花邀约（探求本能 × 性本能）为章节收束提供了罕见的温情瞬间
- **克里斯/厄里斯**：収集本能（抽奖摊敛财）和她作为幸运女神的身份构成巨大戏剧反讽，探求本能（身份之谜）贯穿卷末
