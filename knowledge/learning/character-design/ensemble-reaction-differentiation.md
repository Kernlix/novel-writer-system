# 群像差异化——开篇反应分配法

## 技法分析

《十日终焉》第1章展示了群像塑造的顶级手法：**让所有角色面对同一事件，通过反应差异来立人设**。角色被囚禁在陌生房间中，每个人的应对方式直接暴露其性格内核：

| 角色 | 反应 | 暴露的性格标签 |
|:----|:----|:-------------|
| **齐夏**（主角） | 沉默数人头，"九个人"，摸口袋发现手机被收 | 冷静观察者、细节敏锐、不自来熟 |
| **清冷女人**（章律师） | 用法律术语威胁、计算时间、嫌弃灰尘 | 理性至上、依赖文明规则、独立性强 |
| **花臂男**（乔家劲） | 暴力威胁、骂"冚家铲"、试图武力制服 | 冲动、习惯用暴力解决问题、底层出身 |
| **白大褂医生** | 追问细节"你怎么知道24小时" | 质疑型思维、职业习惯驱动 |
| **健硕年轻人**（楚天秋） | 试图领导、提出"所有人不说谎"方案 | 领导欲强、理想主义倾向、正直外壳 |
| **甜甜** | 性感打扮、自称"技术工作者"、讲述中带大量生活细节 | 底层工作者、用世俗经验应对危机 |

**核心机制**：

1. **每个角色的反应方向不同**：一个推理、一个威胁、一个暴力、一个质疑、一个领导、一个叙事——覆盖了人类面对未知恐惧的六种典型应激模式。
2. **两三句话建立完整初印象**：不需要把简历念一遍，角色通过"怎么说话"（措辞、语气、逻辑方式）完成自我展示。
3. **反应深度反映了社会阶层**：法律术语暗示中产/精英教育背景，脏话暗示底层摸爬滚打，职业细节暗示社会身份。

## 可复用场景

- **开篇群像登场**：主角团第一次集体出场时，让每个人对同一事件做出不同反应
- **危机降临场景**：灾难/意外发生时，群像反应揭示性格
- **新角色加入场景**：新成员进入团队时，通过她/他对现有团队文化的反应立人设
- **决策分歧场景**：团队面临选择，每个角色主张不同方案，通过方案类型暴露价值观

## 自我提升方案

### 对照分析

当前项目"转生深渊领主"的角色出场方式偏描述式（先写外貌身份再写行为），缺少"同一事件的多反应对照"。当前罗姆、维克多、莉莉丝等角色的初登场更多是各自独立场景，没有形成反应矩阵的对比效果。

### 优化措施

在角色档案模板中新增 **"应激反应模板"** 字段：

```
名称：XXX
应激偏好：在未知危险面前第一反应是...
  - 类型：战斗/推理/逃避/领导/质疑/求助/观察
  - 标志性语言：角色在紧张时最常说的话
  - 行为惯性：不经思考就会做的动作
```

同时在团体登场场景中，设计 **"中央事件"** 让所有角色共同面对，写作时遵循三步法：
1. 设计一个所有人都必须回应的刺激（封闭空间、突然袭击、神秘信息）
2. 为每个角色分配不同的反应方向（确保覆盖范围广且不重复）
3. 通过反应的自然冲突推动信息释放（A的推理引来B的质疑，C的行动引发D的反应）

### 应用承诺

在下一次涉及多角色共同出场的场景中，主动设计一个"中央事件"，为至少3个角色分配差异化反应，并记录效果以供后续改进。
