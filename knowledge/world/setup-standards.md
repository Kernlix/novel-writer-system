# 创作设定协调指南

## 角色层面

- **属性模板**：每角色至少包含 姓名/年龄/身份/外貌/性格/动机/内心冲突/成长弧光
- **关系网**：角色之间至少有"认识"以上的关系（利益/情感/血缘/敌对）
- **知识边界**：每个角色知道什么、不知道什么，必须明确记录

## 剧情层面

- **分卷结构**：每卷有独立的核心冲突和解决
- **伏笔管理**：伏笔埋设时需要登记在 `大纲/伏笔追踪.md`
- **节奏控制**：紧张章节后紧跟放松章节，避免持续高密度冲突

## 世界观层面

- **一致性**：力量体系的规则不能中途改变（除非有合理剧情解释）
- **限制**：角色够强≠可以为所欲为，必须有制约机制
- **成本**：使用力量/魔法/系统必须有代价

## 协调检查清单

- [ ] 角色动机与世界观设定不冲突
- [ ] 剧情发展依赖的世界观规则已在前文铺垫
- [ ] 角色能力增长的节奏与剧情需要匹配
- [ ] 新引入的设定不被后续剧情"遗忘"
- [ ] 不同势力的技术水平符合世界基线

## 世界观评估进阶维度

### 【素晴学】框架深度控制
- 数值体系的复杂性是否与叙事价值成正比？每增加一个数值就问：这个数值能否转化为读者的情感体验？
- 如果去掉某个设定元素（等级/经验值/技能树），情节是否仍然成立？如果成立，考虑简化或删除

### 【素晴学】信息释放策略评估
- 设定是一次性倾倒还是分阶段释放？每个新设定片段是否让之前的信息更有趣？
- 设定信息是否被包装在日常/对话/冲突中展示，而非通过旁白说明？
- 读者是否有"同步发现"的体验，还是处于"被单方面告知"的状态？

### 【素晴学】反派生活化评估
- 反派角色是否有"与主线无关"的个人目标或生活？
- 反派阵营的内部动力是什么？除了与主角对立，他们为什么存在？
- 如果去掉反派，这个世界是否还有自洽的生活逻辑？

### 【素晴学】日常锚点评估
- 作品中是否有稳定的日常场景（吃饭/社交/经济/居住）？
- 日常困境是否与史诗情节同等重要地驱动叙事？
- 去掉史诗元素故事还能成立吗？去掉日常元素呢？
