## 二、新增Domain识别（vs V1-V5基线）

### 🔴 Domain 1: 身份悬疑/秘密身份喜剧（Identity Suspense / Secret Identity Comedy）

**判断**: ⚠️ **完全新domain，现有体系无覆盖**

第6卷引入「义贼」作为持续的秘密身份叙事引擎——和真拥有双重身份（冒险者和真 / 假面义贼），且知情者极少（仅达克尼斯+克里斯+爱丽丝?）。这是素晴系列首次使用「角色对自身另一身份的系统性隐藏→多人误认→揭露悬念」作为跨章叙事引擎。V1-V5仅有个别的临时伪装（如和真偶尔冒充他人），从未有「持续整卷的秘密身份」作为核心叙事结构。

| 来源 | ID | 本能核心 |
|:---|:---|:-----|
| ch4 | SN05 | 群像视角「侧写式」战果报告——每个角色通过第三人称侧面呈现，避免主角全知 |
| epi | writer-002 | **五重身份误认叠罗汉**——N-1人不知道+1个镇压者+主角+读者全知的错位结构 |
| epi | writer-003 | **互相甩锅=信息传递**——刑讯场景中两人互咬自然完成事件说明 |
| epi | plot-001 | **六线同步清算**含身份线——义贼身份在卷末仍「半暴露」状态 |
| epi | plot-005 | **惠惠粉丝信**——身份悬疑定时炸弹（崇拜者不知偶像=同居人） |
| epi | char-003 | **达克尼斯「全知重压」**——唯一知情人沉默作为信息防火墙 |
| epi | qa-001 | 义贼装备来源验证——假面=另一枚定时炸弹（物证） |
| ch2 | writer-005 | 义贼宣言→库雷娅反杀——用提案拖延→被对手以「接受+附加条件」反制 |
| ch4 | DL04 | 御剑队伍配置精确预设——身份信息以「贵族八卦」而非旁白形式呈现 |

**≥3条**: ✅ **≥9条本能，远超招募阈值**

**建议招募**:
- **新Agent**: `identity-suspense-writer` (身份悬疑写手 🎭)
  - 职责: 秘密身份叙事构建、多人误认结构设计、身份揭露时机调度、信息不对称管理
- **关联Skill(s)**: `multi-layer-misidentification` (多层误认结构)、`secret-identity-comedy` (秘密身份喜剧引擎)、`reveal-timing-mechanics` (揭露时机力学)

### 🔴 Domain 2: 兄妹/疑似家族关系动力学（Pseudo-Family / Brother-Sister Dynamics）

**判断**: ⚠️ **完全新domain，romance-writer 不覆盖此领域**

和真×爱丽丝的「哥哥-妹妹」关系与和真×惠惠的恋爱CP关系是根本不同的关系类型。此处涉及的是：非血缘疑似家族关系、权力不对等（年上/年下+平民/王族）、「被需要」的情感需求（vs 恋爱中的「相互吸引」）、称呼演化作为关系温度计、以及「哥哥角色扮演」的成瘾→破坏→挽回失败弧线。romance-writer 专精恋爱喜剧感情线（握手→告白→公开确认），不适用于此类疑似家族关系。

| 来源 | ID | 本能核心 |
|:---|:---|:-----|
| ch1 | TM03 | 「想要妹妹」喜剧表层与情感深层——和真在异世界缺乏的「被无条件仰慕/被需要」的关系需求 |
| ch4 | TM02 | **搞笑外壳下「兄妹」情感真核**——三层包装（黄色笑料→冒险新鲜感→孤独者的共情） |
| ch2 | char-001 | **和真「哥哥角色扮演」成瘾→破坏→挽回失败**——完整自我毁灭弧线 |
| ch2 | char-002 | **爱丽丝「逆向成长」**——教育者缺点传递给学生（怀疑→反击→抓破绽） |
| ch2 | plot-005 | **「哥哥」称呼渐进式陷阱**——从无意识→被反复索取→因内裤降温→恶性循环 |
| epi | char-001 | **和真「成为真正的兄长」**——从「想要妹妹（消费）」到「约定再战（尊重）」进化 |
| epi | char-002 | **爱丽丝三重面孔切换**——公主→少女→战略家，远超12岁的叙事成熟度 |
| ch1 | CP04 | 爱丽丝「公主→少女」单场景反转 |
| ch4 | CP04 | 爱丽丝「公主→少女→孤独者」三层剥开 |

**≥3条**: ✅ **≥9条本能，远超招募阈值**

**建议招募**:
- **新Agent**: `pseudo-family-writer` (疑似家族写手 👨‍👧)
  - 职责: 非血缘疑似家族关系构建、兄妹/姐弟/亲子拟态动力学、称呼演化的叙事化使用、权力不对等亲密关系写作
- **关联Skill(s)**: `brother-sister-dynamics` (兄妹关系力学)、`title-evolution-as-thermometer` (称呼作为关系温度计)、`reverse-growth-arc` (逆向成长弧线——教育者缺点传递)

### 🟡 Domain 3: 身体交换身份喜剧（Body-Swap Identity Comedy）

**判断**: ⚠️ **边界domain——部分可吸收进 humor-writer，但「身份交换+语言同步+性别错位」的复合技法独特性强**

第4章通过神器「交换身体」创造了素晴系列首次大规模身体交换喜剧。这不同于一般的「变身/伪装」——身体交换涉及：语言同步默认规则、性别特征感知错位、交换中的死亡风险（严肃化转折）、以及「在他人身体中体验另一阶级/性别的日常」的伦理喜剧。

| 来源 | ID | 本能核心 |
|:---|:---|:-----|
| ch4 | DL05 | **交换身体「语言同步」默认规则**——声音/外貌与语气的持续错位=笑点 |
| ch4 | ER04 | **内衣无声叙事+更衣室恐慌**——男性思维解释女性内衣逻辑=黄色凝视包装为推理 |
| ch4 | DL01 | 神器「削弱版」设定作为叙事安全阀——时间限制+交换中死亡不可逆 |
| ch4 | SP04 | 神器事件「三幕悬疑」闭合——同一道具三种情感色调（喜剧→搞笑→惊悚） |

**≥3条**: ✅ **4条本能，达阈值**

**建议**: 两种处理方式：
- A. 吸收进 `humor-writer` Agent，新增「身体交换喜剧」子模式
- B. 作为 `identity-suspense-writer` 的「身份模糊」子模块

**倾向: A（吸收进 humor-writer）**——身体交换本质仍是喜剧引擎，且本能数刚好达阈值。

### 🟡 Domain 4: 阶级叙事/贵族-平民喜剧（Class Narrative / Noble-Commoner Comedy）

**判断**: ⚠️ **新domain，但本能分散度较高，可部分吸收**

V1-V5中达克尼斯的贵族身份主要用于角色搞笑（受虐狂+大小姐反差），但V6首次将「阶级」作为系统性的叙事张力轴——王城空间的阶级分层、贵族面具崩塌、平民在权力空间中的边缘化、阶级距离的不可逾越性。这不是「某个角色是贵族」的设定，而是「空间本身就是阶级叙事」的技法。

| 来源 | ID | 本能核心 |
|:---|:---|:-----|
| ch1 | TM04 | **阶级叙事「傲慢→打破→平等」三幕**——打脸（物理+比喻）瓦解阶级隔阂 |
| ch4 | TM04 | 「平民冒险者」与「贵族公主」阶级距离感——三次空间隔离（被带走→特别席→独自一人） |
| ch4 | SC03 | **庆功宴会「中心→边缘」同心圆权力空间**——空间本身就是阶级叙事 |
| ch2 | char-004 | **达克尼斯晚餐会面具六段崩塌**——贵族人设在公众场合的完整崩解弧线 |
| ch2 | char-003 | 库雷娅从「暴怒机器」到「官僚柔术家」——战术层次=贵族教育产物 |
| epi | scene-001 | 王城告别剧场——多角色在权力空间（王城上层）中轮流登台 |

**≥3条**: ✅ **≥6条本能**

**建议**: 两种处理方式：
- A. 新增 `class-comedy` Skill（阶级喜剧技法），归属 humor-writer
- B. 吸收进 `worldbuilding` Skill 的「社会结构」子模块 + humor-writer

**倾向: A（新增Skill）**——阶级叙事技法与worldbuilding（设定层面）和humor（搞笑层面）都有交叉但核心独立，单独成Skill更清晰。命名为 `class-comedy` 或 `power-space-narrative`。

### 🟢 Domain 5: 物品多义性叙事（Multi-Definition Object Narrative）

**判断**: ✅ **可吸收进现有体系**

「戒指」在第6卷卷末作为「三方定义装置」（和真=赃物/达克尼斯=国家机密/爱丽丝=婚约信物）是全卷终极的多义性技法。同一物品在不同角色眼中具有完全不同的定义，且三种定义同时为真。

| 来源 | ID | 本能核心 |
|:---|:---|:-----|
| epi | plot-002 | **戒指三方定义**——同一物品=赃物vs国家机密vs婚约信物 |
| epi | writer-001 | 戒指「婚约信物」三重升级揭露——从偷窃笑点→达克尼斯惊恐→爱丽丝独白 |
| epi | scene-002 | 五件物品（沙发/红茶/酒瓶/粉丝信/戒指）定义「家」的情感地图 |

**≥3条**: ✅ **3条本能，刚好达阈值**

**建议**: 吸收进 `worldbuilding` Skill 或 `plot-outline` Skill，新增「多义性物品」子模块。

## 四、部署计划

### 立即部署 (P0)

#### 1. 新Agent: `identity-suspense-writer` (身份悬疑写手 🎭)

- **职责**: 秘密身份叙事构建、多人误认结构设计、身份揭露时机调度、信息不对称管理、身份信息在喜剧与悬疑之间的平衡
- **触发条件**: 章节涉及角色拥有「不为部分/全部同伴所知的第二身份」
- **关联Skill (3个)**:
  - `multi-layer-misidentification` — N-1层不知道+1个镇压者+主角+读者全知的多层误认错位结构
  - `secret-identity-comedy` — 秘密身份作为喜剧引擎（全员夸义贼·本人不能说）
  - `reveal-timing-mechanics` — 身份揭露时机的跨卷管理（半暴露状态、定时炸弹预埋）

#### 2. 新Agent: `pseudo-family-writer` (疑似家族写手 👨‍👧)

- **职责**: 非血缘疑似家族关系构建、兄妹/姐弟/亲子拟态动力学、称呼演化叙事化使用、权力不对等亲密关系写作、「被需要」心理需求的喜剧-情感双层包装
- **触发条件**: 章节涉及「非恋爱/非友情的第三类亲密关系」（如兄妹拟态、收养关系、师徒拟似亲子）
- **关联Skill (3个)**:
  - `brother-sister-dynamics` — 兄妹关系力学（成瘾→破坏→挽回失败弧线 + 逆向成长）
  - `title-evolution-thermometer` — 称呼演化作为关系温度计（升温→降温→恶性循环）
  - `found-family-emotional-core` — 「被需要的需要」情感真核的喜剧双层包装

### 后续迭代 (P1)

#### 3. humor-writer 扩展: 「身体交换喜剧」子模式
- 吸收 Domain 3 的4条本能
- 新增子模式：语言同步默认规则、性别特征错位、交换中的严肃化转折

#### 4. 新Skill: `class-comedy` (阶级喜剧)
- 归属: humor-writer Agent 或独立
- 触发条件: 章节涉及贵族/平民阶级互动、权力空间分层
- 核心技法: 阶级面具崩塌弧线、空间作为阶级叙事、权力同心圆结构、「傲慢→打破→平等」三幕

### 后续迭代 (P2)

#### 5. plot-outline Skill 扩展
- 「卷末多段收束」模块（Domain 6）
- 「物品多义性」子模块（Domain 5）

#### 6. worldbuilding Skill 扩展
- 「食物体系」子模块（啾叫煎蛋跨卷验证）
- 「多义性物品」子模块（Domain 5）
