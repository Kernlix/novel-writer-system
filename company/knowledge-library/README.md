| 👤 角色设计 | `knowledge/character/` | 角色状态管理、弧光设计 + 所学技法 |
| 📖 剧情设计 | `knowledge/plot/` | 伏笔管理、节奏控制 + 所学技法 |
| 🌍 世界观 | `knowledge/world/` | 世界观一致性、力量体系、设定标准 |
| ⚡ 短故事 | `knowledge/short-story/` | 短故事结构与平台适配 |
