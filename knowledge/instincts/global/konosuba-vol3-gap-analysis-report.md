## 🟢 核心结论：无需新增 Agent，需新增 1 个 Skill + 扩充 4 个 Skill

第3卷的喜剧本能绝大多数可以归属已有 **humor-writer Agent**——不存在需要独立 Agent 的新 domain。
但有一个**全新子域**（体制/法庭喜剧）达到 ≥7 条无覆盖本能，远超 ≥3 条阈值，需新增 Skill。

| 缺口子域 | 严重程度 | 本能数 | 建议行动 | 优先级 |
|:---------|:--------|:------|:---------|:------|
| **体制/法庭喜剧**（System/Courtroom Comedy） | 🟡 中度 | 7+ | 新增 `system-comedy` Skill 归入 humor-writer | P1 |
| **双声轨附身喜剧**（Dual-Voice Possession Comedy） | 🟡 轻度 | 4+ | 扩充 `comedy-pattern-library` 新增 §10 模块 | P1 |
| **身份伪装/阶级权力喜剧**（Identity Disguise Comedy） | 🟡 轻度 | 4+ | 扩充 `comedy-scene-design` 新增模块 | P1 |
| **道具喜剧弧线**（Prop Comedy Arc） | 🟡 轻度 | 4+ | 扩充 `comedy-scene-design` 新增模块 | P2 |
| **反派魅力/解构型敌人**（Deconstructed Villain） | 🟢 已部分覆盖 | 3+ | 扩充 `worldbuilding` Skill（与第2卷P2行动合并） | P2 |

| 指标 | 数值 |
|:-----|:-----|

| 阈值判断 | 🔴 **≥3 条，domain 归属 humor-writer → 新增 Skill 而非 Agent** |
| 前2卷是否出现过 | ❌ **第1-2卷仅第2卷终章有「颠覆国家罪」雏形，未形成完整方法论** |

### 代表性 instincts（均无法被现有 4 个 Skill 的方法论覆盖）

| ID | 技法 | 章节 | 为什么现有 Skill 不覆盖 |
|:---|:-----|:-----|:---------------------|
| `konosuba:v3ch1-w04` | **测谎铃作为多功能叙事道具**——一个道具同时承担情节驱动力、喜剧制造器、主题隐喻三层功能 | 第1章 | 现有 prop 相关技法散见于各 Skill 但无「体制道具」专项——测谎铃的核心笑点来自「体制对个体的审查」而非单纯的反差 |

| `konosuba:v3ch4-writer-001` | **"炫耀性说漏嘴"三段式揭罪结构**——罪魁祸首阿克娅不是被揭发，而是因邀功自我暴露：「多亏有我，魔法阵还在！」→ 不是认罪是炫耀 | 第4章 | 现有 defect-comedy-engine 覆盖「缺陷驱动笑点」但不覆盖「被体制追责时的无自觉自我暴露」——关键在于「角色真心认为自己在帮了忙」而非「犯了错」 |
| `konosuba:v3ch4-writer-002` | **瑟娜作为"道德压力+关键道具"双向叙事驱动轮**——检察官既是良心的拷问者（制造内疚），又是剧情的强制推动者（提供封印符咒），且「正直至不知变通」的性格让每次互动都产生无心助攻 | 第4章 | 现有 skills 无「体制角色作为双向驱动轮」的设计方法论——体制代言人既是障碍又是暗中助力者 |
| `konosuba:v3ch1-r01` | **"被冤枉"的共情锚点**——和真拯救了城镇却被指控为恐怖分子，直击「好人没好报」的普遍性焦虑，读者从第一页就代入被冤枉感 | 第1章 | 现有 skills 不覆盖「用体制不公制造读者情感绑定」——这不是角色间的情感驱动，而是「个体vs体制」的正义感激活 |

### 现有覆盖情况

| Agent/Skill | 与体制/法庭喜剧的关系 |
|:------------|:---------------------|
| `comedy-scene-design` | 覆盖：反高潮、反差四段式、缺陷引爆链。**不覆盖**：体制场景类型混搭、严肃空间解构、审判节奏设计 |
| `comedic-dialogue` | 覆盖：漫才对话、吐槽节奏。**不覆盖**：法庭问答格式、「辩护=揭短」的对话反转、检察官-嫌疑人权力不对等对话 |
| `defect-comedy-engine` | 覆盖：缺陷设计、标签化、接力碰撞。**不覆盖**：「缺陷在被体制审查时的暴露」——同样的缺陷（阿克娅的KY）在法庭场景中被系统性地逐一揭发 |
| `comedy-pattern-library` | 覆盖：9 种高级喜剧格式。**不覆盖**：体制/法庭类型混搭格式 |

**关键差距**：现有 4 个 Skill 都能处理「角色与角色之间的喜剧」，但都无法处理「角色与体制（法律/审判/官僚系统）之间的喜剧」——后者需要全新的方法论：体制场景的类型混搭、严肃空间解构、体制代言人的双向叙事功能、被冤枉的共情锚点设计。

### 行动建议

```
🟡 新增 Skill：system-comedy（体制喜剧方法论）
   - 归属：humor-writer Agent
   - 职责：法庭/审判/官僚体制场景的喜剧构造——
           ① 体制场景类型混搭（法律×异世界/中世纪×现代法庭）
           ② "严肃空间解构"技法（将法庭/议会/审判的仪式性喜剧化）
           ③ 体制代言人角色设计（检察官/法官/官僚的双向叙事功能）
           ④ "被冤枉"共情锚点的激活与释放节奏
           ⑤ "辩护=伤害"的体制特有对话反转格式
           ⑥ 测谎/审讯/证据等体制道具的多功能设计
   - 覆盖 7+ 条体制喜剧 instincts
   - 适用场景：审判章、听证会、被冤枉/追责、体制与个体的碰撞
```
