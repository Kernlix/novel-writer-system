# 资源经济检查标准

## 核心原则

数字会说话：人口、资源、战力的定量关系必须自洽。读者可能不会逐个数，但明显荒谬的数字会破坏沉浸感。

## 检查清单

### 人口与经济
- [ ] 城市/国家的规模与人口匹配
- [ ] 军队数量与总人口的比例合理（古代军队通常不超过总人口的1-3%）
- [ ] 粮食产量能支撑城市人口和军队消耗
- [ ] 税收/经济来源清晰，没有"钱从天上掉下来"
- [ ] 战争/灾难后的恢复速度合理

### 修炼/升级体系
- [ ] 从低阶到高阶的所需资源/时间呈指数增长（而非线性）
- [ ] 高阶角色的数量占比符合金字塔结构
- [ ] 修炼资源的分布与势力格局匹配（有资源的地方必然有争夺）
- [ ] 突破瓶颈的设定清晰：需要什么条件，多久一次

### 战斗/战力核算
- [ ] 不同等级角色的战力差距有明确的比例参考
- [ ] 多人围攻的效果合理（不是100个低阶等于1个高阶的简单加法）
- [ ] 战斗消耗（体力/魔力/精神力）的恢复速度合理
- [ ] 装备/道具的获取难度和效果匹配

### 时间与距离
- [ ] 不同地点之间的移动时间与交通方式匹配
- [ ] 信息传递的速度与通讯手段匹配
- [ ] 任务的完成期限合理（不可能三天完成一个月的路程）

### 【素晴学】资源循环叙事效能检查
- [ ] 经济系统是背景板还是剧情引擎？去掉经济元素剧情是否仍然成立？
- [ ] 角色的经济困境是否产生不可回避的选择（而非仅仅是增添不便）？
- [ ] 经济收益与损失是否有连锁反应（赚钱→搞破坏→赔更多→更穷的循环）？
- [ ] 是否满足了"需求刚性、收支不平衡、有不可预期的支出"三大特征？
- [ ] 角色的经济状况是否影响其社交/声誉/冒险选择？（非独立的经济系统）
