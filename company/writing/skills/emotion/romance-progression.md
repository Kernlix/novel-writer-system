# 🤍 感情线渐进四阶段模型 Skill

> 基于素晴第5卷提炼的 CP 关系渐进写作模型。适用于恋爱喜剧中男女主角关系从暧昧到确立的完整弧线。

## 核心模型：四阶段递进

```
阶段1: 私下暧昧（Private Ambiguity）
  → 身体靠近 + 语言暗示 + 场景留白
  → 例：黑暗中握手→秒睡（反高潮中断）

阶段2: 口头告白（Verbal Confession）
  → 直球告白 + 可撤回语尾 + 傲娇包装
  → 例：「我喜欢和真哦？」（哦？=可撤回）→「想要优秀的魔法师吗？」（推销包装）

阶段3: 公开宣告（Public Declaration）
  → 对第三方宣告 + 第三方震惊反应 + 社会化验证
  → 例：「请你们不要朝我的男人暗送秋波」——措辞过度（炫耀）

阶段4: 行动回应（Action Response）
  → 不靠台词而靠行动确认 + 沉默>语言 + 同伴「违背」=最深理解
  → 例：偷改冒险者卡片全投爆裂魔法 = 「你就是你，不需要改变」
```

## 工作流程

1. **识别当前阶段**: 从已有章节中定位CP关系处于哪个阶段
2. **规划下一阶段**: 确定本章/本卷应推进至哪个阶段（不可跳级）
3. **设计推进场景**: 为阶段转换创建专门场景（密室/散步/战场/被窝）
4. **选择技法**: 从技法库中选择适合当前阶段转化的技法
5. **设置反高潮**: 每个阶段末尾应有一个「打断/留白」作为情感张力调节

## 阶段转化技法库

### 1→2（暧昧→告白）
- 反高潮告白：建立期待→内心过山车→睡眠/外力打断
- 竖排单字排版：用视觉减速暗示「一字一顿」的真挚
- 黑暗场景+触觉描写：消除视觉、强化触感的亲密空间
- 双段告白拆解：直球→推销包装（角色自我定义=告白核）

### 2→3（告白→公开）
- 第三方震惊验证：「旧人设见证者」的震惊=成长的证明
- 措辞过度：用「我的男人」而非「男朋友」传达占有欲/炫耀
- 朋友逃跑后的嗤笑：宣告者从「不关心社交」到「享受社交胜利」

### 3→4（公开→行动）
- 行动替代台词：用「替对方做决定」替代「我也喜欢你」
- 违背=尊重悖论：违背表层意愿=成全深层意愿
- 复杂表情验证：无奈+喜悦+释然的三重表情=被理解
- 延迟揭露：操作→沉默→安排揭露时机→共同发现真相

## 关键原则

1. **不可跳级**: 不能从阶段1直接跳到阶段4
2. **每次推进应有回撤**: 告白后要有害羞逃跑→下次再见面的尴尬→打破尴尬的契机
3. **场景同步收紧**: 从公开空间（街道/战场）→ 半私密（房间）→ 绝对私密（被窝）
4. **第三方反应 > 两人对话**: 关系的「外部验证」比「内部确认」更有说服力
5. **反高潮 = 节奏控制器**: 在读者期待「终于要成了」时打断，维持长线张力

## 输出

- 本章CP关系推进方案（当前阶段→目标阶段+场景设计+技法选择）
- **感情线节奏日历**（跨章节的阶段分布图，输出到 `大纲/感情线追踪.md`）
