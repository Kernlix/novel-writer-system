# ⚖️ 两难卡诺图 Skill (Dilemma Karnaugh Map)

> **核心来源**：《十日终焉》第470章——乔家劲面对地马的最后一击（铁球）：如果接，铁球与球棒碰撞产生火花引燃地面油层→全员被大火吞噬；如果不接，铁球直接击中车上队友→队友死亡。两个选项都通向死亡，但「自身的死亡」和「队友的死亡」是不同的代价。
> 
> **提炼方法**：从生死张力、策略与博弈、人性试炼维度提取的「决策困境设计」叙事技法
> 
> **适用场景**：故事高潮中的终极抉择、角色成长的关键转折点、展示角色的价值观优先级

---

## 📋 技法定义

**两难卡诺图**是指在故事中设计一个所有选项都有致命代价的决策困境（如数字电路中的「卡诺图」简化逻辑——所有输入组合都导致输出为「真」）。角色在面对这种困境时，无论选择哪个选项，都要承受某种不可逆的损失。

**关键设计原则**：
1. **所有选项都有代价**（没有安全的选项）
2. **代价不同质**（自身受伤 vs 他人死亡 vs 原则背叛——让「取舍」有意义）
3. **读者能理解两个代价**（不需要额外解释每个后果）
4. **角色必须做出选择**（不能逃避或绕开困境）

---

## 🎯 技法原理

### 心理机制

两难卡诺图的叙事效果源于读者的「替代决策焦虑」——当读者站在角色的角度时，自己也无法做出「正确」的选择。这种身份的代入感让读者不是旁观者，而是「决策者」。当角色最终做出选择时，读者感受到的不是「正确答案」的满足，而是「牺牲」的情感重量。

### 与其他困境的区别

| 类型 | 设计 | 效果 | 示例 |
|:----|:-----|:-----|:-----|
| **简单困境** | 一个好选项+一个坏选项 | 读者期待角色选好的 | 躲开攻击 vs 被击中 |
| **道德困境** | 两个选项有不同道德成本 | 读者思考「什么是对的」 | 说谎救人 vs 诚实害人 |
| **两难卡诺图** | 两个选项各有致命代价 | 读者感受「牺牲的重量」 | 接=火海 vs 不接=队友死 |

## 📝 写作步骤

### 步骤1：建立两个选项

构筑两个在逻辑上成立的选项，每个都必须有明确的致命后果：

```
✅ 正确示例（第470章）：
  选项A：接铁球 → 金属碰撞产生火花 → 引燃地面黑油 → 全员火海
  选项B：不接铁球 → 铁球继续飞行 → 击中车上队友 → 队友死亡

❌ 错误示例：
  选项A：接铁球 → 可能会受伤
  选项B：不接铁球 → 队友可能会受伤
  → 「可能」「可能会」削弱了两难的命定性
```

**设计公式**：
```
选项X = 行动 + 物理机制 + 后果（全局/不可逆）
选项Y = 不行动 + 物理机制 + 后果（全局/不可逆）
```

### 步骤2：确保「理解成本」最低

读者应当在看到困境的瞬间就能理解两个后果，不需要角色或叙事者解释：

```
✅ 正确示例（依赖物理常识）：
  铁球+铁球棒=火花
  火花+油层=大火
  大火=死亡
  → 读者通过物理常识自动完成推理链

❌ 错误示例（依赖虚构设定）：
  如果接球，魔法的反噬力会撕裂空间
  如果不接球，诅咒之力会吞噬灵魂
  → 读者需要额外的设定说明才能理解代价
```

**最佳材料来源**：日常生活的物理常识（油+火=燃烧、高空坠落=死亡、溺水=窒息）

### 步骤3：压缩决策时间窗口

不给角色思考时间——在极短的时间内做出选择：

```
✅ 正确示例：
  「球已经来到了眼前。」
  → 乔家劲没有时间充分思考，几乎凭本能做出决定

❌ 错误示例：
  「乔家劲思考了很长时间，反复权衡利弊……」
  → 决策的时间优势会削弱「牺牲」的悲壮感
```

### 步骤4：做出选择并承担后果

角色做出选择后，立即展示选择带来的后果——让读者看到「牺牲」的实质性：

```
✅ 正确示例：
  乔家劲接球 → 火花→大火→全员着火
  → 后果立即兑现，没有延迟
  
  然后立即展示「灭火」过程
  → 选择之后是代价的支付，而不是轻描淡写的解决
```

## 🎬 应用示例

### 示例1：救援困境

```
（情境：洪水泛滥，A在水中抓住木板，B在远处呼救）
选项A：抓住木板自保 → B被洪水冲走
选项B：放弃木板游向B → 体力不支，两人都可能淹死
（抉择：「我不是救你，我只是不想一个人漂着。」A松开了木板……）
```

### 示例2：资源困境

```
（情境：急救包只剩一剂抗生素，两个伤员都需要）
选项A：给年长者 → 年轻人伤口感染
选项B：给年轻人 → 年长者抵抗力差可能撑不过
（抉择：「给那个还有孩子的人。」——这不是医疗决策，而是价值观决策）
```

## ⚠️ 常见陷阱

| 陷阱 | 问题 | 解决方法 |
|:----|:-----|:---------|
| **代价同质** | 两个选项都是「自己受伤程度不同」 | 让一个选项影响自己，另一个影响他人 |
| **隐藏的第三选项** | 角色或读者找到「两全其美的办法」 | 在困境设计阶段就堵死所有避重就轻的可能 |
| **代价太抽象** | 「失去希望」「信念崩塌」等抽象代价 | 使用物理上的、可感知的代价（烧伤/骨折/死亡） |
| **选择后代价被取消** | 角色做出牺牲但最后奇迹般无损失 | 牺牲必须是真实的、不可逆的——否则困境是虚假的 |
| **情感铺垫不足** | 读者对角色的困境没有情感投入 | 在困境前建立角色与团队/目标的情感联系 |

## 💡 变体

### 变体1：假性两难（实际有第三选项）
角色以为只有两个选择，但在最后一刻发现隐藏的选项。使用时要小心——这会让之前的困境显得「廉价」：
> 「等等……油层只在地面，如果把车子抬起来？」

### 变体2：逐步升级的两难
困境不是一次性给出，而是逐步收紧：
> 第一层：接球or不接球
> 第二层：即使接了球也可以选择方向
> 第三层：但任何方向都有代价

### 变体3：旁观者的两难
不是主角，而是旁观者面临两难——是否告知主角真相：
> 你知道主角的战术有致命缺陷，但告诉他真相可能让他分心死得更快。选择沉默还是告知？

---

## 🔗 关联技法

- **虚假安全感**：可以在两难前建立短暂的「安全期」，增强困境的冲击力
- **对手逆转援助**：可以在两难困境后，由意想不到的第三方提供援手
- **能力双刃剑**：两难中的选项之一可以涉及能力副作用的激活
