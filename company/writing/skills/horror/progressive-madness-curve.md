# 渐进式疯狂曲线（Progressive Madness Curve）

> 「好有意思呀……就好像在看静了音的电视。」——苏闪·第403章

## 概述

让一个角色逐步失去理智的过程本身就是微型叙事弧线。本技法将「疯狂」分解为六个可操作的渐近阶段，每个阶段都在前一个阶段的基础上叠加更不合理的元素，让读者在「她只是累了—她是不是看见了什么—她是真疯了—她真的要死了」的认知摇摆中逐步接受角色的死亡结局。

核心原则：**角色在疯狂的前90%时间仍然保持理性的语气和句式**——让读者无法判断「她是疯了还是真的看到了」。

---

## 技法结构

### 六阶段渐进崩溃曲线

```
阶段1 ── 轻度异常（Mild Anomaly）
  角色开始看到/感知到其他人看不到的东西
  语气仍然正常：「你们发现了吗？人越来越多了」
  → 读者反应：什么事？她看花眼了吧？
  
阶段2 ── 感知深化（Deepened Perception）
  角色能够描写异常事物的细节（颜色、形状、光芒）
  语气仍然是理性描述：「你的光芒很温暖……就像太阳般温暖」
  → 读者反应：说得这么具体……不会是真的吧？
  
阶段3 ── 认知脱节（Cognitive Disconnect）
  角色说出其他人无法证实也无法证伪的断言
  内容开始与周围人的感知不同步
  「我们被人包围了」
  → 读者反应：队友说没看见→她真出问题了
  
阶段4 ── 幻觉互动（Hallucination Interaction）
  角色开始与不存在的人/物对话
  对话中仍保持正常的日常句式
  「你为什么不动呢？」——对路灯说话
  → 读者反应：她已经不在我们这个世界了
  
阶段5 ── 感官封闭（Sensory Shutdown）
  角色对物理世界的感知开始关闭
  对其他人说话的声音失去反应
  给出文学性的感知描述：「好像在看静了音的电视」
  → 读者反应：她快走了

阶段6 ── 超现实→死亡（Surreal → Death）
  最后、最致命的幻觉（「天上飘着一个女人」）
  关键生理标记（「眸子变成纯黑」）
  死亡（通常在描写之后紧接确认）
  → 读者反应：那个幻觉是真的吗？还是只是最后的疯狂？
```

---

## 执行要点

### 1. 理性外壳是关键

每一阶段的描写都必须符合以下原则：
- **句式正常**：角色不使用破碎的语法或疯狂的语序
- **语气平静**：角色对自己看到的东西感到好奇而非恐惧
- **语言有文学性**：角色的感知表达具有意象品质（「黑色灯塔」「静音的电视」）

### 2. 生理标记同步

随着疯狂加深，使用连续的生理标记来同步可视化进程：
- 眼睛颜色变化（如：血红色→暗红色→纯黑色）
- 这是「人类意识熄灭过程」的视觉符号

### 3. 队友的反应作为旁证

在每个阶段，通过其他角色的反应来「确认」异常的可信度：
- 第一阶段：队友不在意（「你回响过度了」）
- 第二阶段：队友开始担忧（「苏闪，你没事吧？」）
- 第三阶段：队友开始讨论要不要干预（「她没救了」）
- 第四阶段：队友准备动手（「我去杀了她吧」）

### 4. 最后幻觉的悬置

在角色死亡前，给她最后一个、最惊人的幻觉——这个幻觉应该被设计为「可能是真的也可能是最后疯狂」的模糊状态。它让读者永远无法确定角色在最后时刻看到的是「真相」还是「疯狂」，给死亡的结局留下一个永恒的谜。

## 使用模板

```
【阶段1】"你们发现了吗？"角色用正常的语气说，"那里有几个奇怪的东西。"
其他人顺着他看的方向看去——"没东西啊。"
"怎么可能？明明就在那里。"

【阶段2】"不是普通的影子……它们会发光。"角色的语气仍然理性，"那个蓝色的发光体很像一个人的轮廓。"

【阶段3】"越来越多了。"角色的声音仍然平静，"到处都是。我们被包围了。"
队友开始后退——不是因为看见了东西，而是因为角色本身的异常状态可怕。

【阶段4】角色走到一个无人的角落。"你好。你也是来找我的吗？"
队友面面相觑："他在和谁说话？"

【阶段5】角色停止回应队友的呼唤。他眼睛睁着但瞳孔已经微散。
"我听不见你们的声音了。就像是……整个世界被按下了静音键。好安静。好美。"

【阶段6】角色突然睁大双眼——"上面！上面有人！"
这是他说完的最后一句话。眸子/肤色/状态发生关键性变化。
倒地。死亡。
```

## 应用场景

| 适用类型 | 说明 |
|:---------|:-----|
| 超能力代价展示 | 能力使用过度导致精神崩溃 |
| 恐怖氛围高潮 | 角色「渐入疯狂」作为长期的恐怖线索 |
| 配角死亡戏 | 让配角的死亡不再只是一个「死了」的简讯，而是完整的情感弧 |
| 世界观揭示 | 角色在死亡前看到的「最后幻觉」可能是系统的真实面貌 |
