# 天灾打破博弈叙事

> **核心来源**：《十日终焉》第201-210章——天灾介入打破博弈格局
> **提炼方法**：从天灾叙事、世界观转换、犯罪同盟背叛、全员猎物相关维度中提取可复用叙事技法
> **适用场景**：世界观转换、外部危机介入、犯罪同盟瓦解、格局升级类章节

---

## 技法1：外力介入打破博弈

自然灾害/外部事件作为**不可控变量**介入，让所有精心策划的博弈瞬间失效。

### 1.1 打破效果

| 博弈类型 | 天灾前 | 天灾后 |
|:---------|:-------|:-------|
| **结盟** | 精心策划的联盟 | 自然灾害让联盟失去意义 |
| **背叛** | 有计划的背叛 | "敌人"和"队友"的区分消失 |
| **算计** | 长线布局 | 一切归零，只能求生 |
| **优势** | 积累的资源/地位 | 在天灾面前毫无价值 |

### 1.2 打破的层次

```
第一层：信息打破
  → 突如其来的信息使所有博弈策略失效

第二层：秩序打破
  → 外部危机让原有社会秩序/规则失效

第三层：认知打破
  → 角色意识到自己的"博弈"在更大危机面前毫无意义
```

### 1.3 写作要点

1. **突然性**：天灾必须出乎角色和读者预料——打破预期
2. **公平性**：天灾对所有人一视同仁——没有人能幸免
3. **不可逆**：天灾造成的改变无法恢复——永久性转折

## 技法2：世界观转换

天灾作为**优雅的世界观转换方式**——从一种故事类型平滑过渡到另一种。

### 2.1 转换类型

| 原始类型 | 转换后类型 | 转换方式 |
|:---------|:-----------|:---------|
| 心理博弈 | 生存冒险 | 外部危机取代内部博弈 |
| 悬疑推理 | 动作冒险 | 真相不再是首要问题 |
| 政治权谋 | 战争求生 | 权力斗争让位于生存需求 |
| 智力对决 | 团队合作 | 个人智力让位于集体行动 |

### 2.2 转换的叙事功能

1. **节奏刷新**：防止读者审美疲劳——故事类型变化带来新鲜感
2. **角色展示**：新环境下角色展现不同面向——深度增加
3. **主题深化**：从"人与人斗"升级到"人与自然斗"——格局扩大
4. **伏笔激活**：天灾可能激活之前埋下的伏笔——前后呼应

### 2.3 转换的过渡技巧

```
【天灾前】精心策划的博弈场景
【天灾征兆】环境异常/角色不安
【天灾降临】不可抗力介入
【天灾后】新规则、新目标、新联盟
```

## 技法3：犯罪同盟的终极背叛

当生存压力增大，**犯罪同盟内部必然出现背叛**——每个人都只为自己。

### 3.1 背叛的递进

```
阶段1：表面合作
  "我们是一条船上的人。"
  → 在危机面前暂时团结

阶段2：资源竞争
  "食物不够所有人分。"
  → 从合作转向竞争

阶段3：公开背叛
  "对不起，但我需要多活一天。"
  → 直接伤害盟友求生

阶段4：终极背叛
  "我不是背叛你。我只是在保护自己。"
  → 将背叛合理化
```

### 3.2 背叛的逻辑

- 犯罪同盟的基础是**利益**，不是信任
- 当利益分配不均或生存压力增大，同盟必然瓦解
- 每个人都认为自己"应该"多活一天

### 3.3 叙事张力

最讽刺的是：**他们在天灾前的博弈，正是导致天灾后无法合作的原因**

## 技法4："黑吃黑"——全员猎物

在天灾介入后，**每个参与者既是猎人也是猎物**——没有人是安全的。

### 4.1 猎物状态

| 角色 | 猎人状态 | 猎物状态 |
|:-----|:---------|:---------|
| **组织者** | 掌控局面 | 天灾面前失控 |
| **执行者** | 执行命令 | 被天灾威胁 |
| **受害者** | 被动承受 | 天灾面前人人平等 |
| **旁观者** | 安全观察 | 天灾波及所有人 |

### 4.2 猎人猎物切换

```
A是B的猎人（A在操控B）
  → 天灾介入
  → A和B都成为天灾的猎物
  → A和B必须合作求生
  → 天灾过后，谁是猎人谁是猎物？
```

### 4.3 叙事效果

1. **格局升级**：从"小博弈"到"大求生"——故事规模扩大
2. **角色平等**：天灾面前人人平等——打破权力等级
3. **主题深化**：从"人与人的恶"到"人在自然面前的渺小"
4. **新关系**：天灾催生新联盟——与之前的人际关系形成对比

### 天灾转折章节的技法组合

