# 🤝 极简信任确认 (Minimal Trust Confirmation)

> "我正在做。"——四个字，承载了身份确认、忠诚确认和信任纽带的全部重量。

## 核心原理

在高压环境中，用4-10个字完成信任关系的确认——不解释、不汇报、不详述。省略的内容越多，信任的重量越大，因为只有在完全信任的关系中，才敢省略所有解释。

### 效果
- 将信任的确认压缩到极致——读者通过"他只说了四个字"来感受情感重量
- 省略本身就是信任的证明——不需要解释，因为对方会懂
- 极简表达在高压环境中更有力——废话是安全环境的特权

## 技法结构

```
阶段1: 铺垫（前文建立角色之间的信任关系/约定/任务）
  ↓
阶段2: 重逢/确认时刻（两个角色在极端环境中短暂相遇）
  ↓
阶段3: 极简表达（一方用4-10字确认身份/忠诚/执行状态）
  ↓
阶段4: 对方理解（不需要追问/确认——信任关系使得省略成立）
  ↓
阶段5: 分离（短暂相遇后立即分开——没有时间多说）
```

## 使用指南

### 适用场景
- 久别重逢但没有时间详谈的场景
- 高压逃亡/战斗中的短暂相遇
- 角色确认自己在执行对方交代的任务
- 信任关系的"最终确认"——在可能永远见不到之前

### 注意事项
1. **必须有前文铺垫**：极简表达的力量来自前文的积累——没有铺垫的极简只是空洞
2. **省略的内容必须是读者能自行补全的**：如果读者不知道"做"的是什么，极简就失去了意义
3. **不宜连续使用**：一次极简确认是核弹，两次就是哑炮
4. **后续应该有分离/中断**：极简确认后不应有长篇对话——那会稀释极简的力量

### 变体
- **行动确认型**：不说"我在做"，而是展示一个已完成的动作（如展示伤痕/成果）
- **沉默确认型**：不说话，只是点头/对视——比语言更极简
- **代号确认型**：用一个代号/暗号替代完整表达（如只说"针"）

## 来源章节分析

**第538章「我正在做」**：齐夏在天马时刻的逃亡中遇到地虎。地虎脸上带伤、虎牙断裂，但只说了四个字"我正在做"——确认自己是齐夏埋下的"针"，确认针正在行动，确认信任没有被辜负。

**关键设计**：地虎的犹豫（"组织了半天措辞"）+ 极简表达（"我正在做"）= 身份错位的尴尬（地级生肖与参与者之间的信任关系）被极简地表达出来。

## 本能提炼

| 本能 | 适用范围 |
|:-----|:---------|
| 信任不需要证明——省略本身就是证明 | 所有信任确认场景 |
| 极简表达在高压环境中更有力——废话是安全环境的特权 | 高压逃亡/战斗中的对话 |
| 名字是最后的尊严——赴死前要求被记住名字是对存在意义的最终确认 | 多人牺牲场景的人性深度 |
