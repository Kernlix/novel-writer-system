# 🔺 团队决策三角悬念 Skill

> **核心来源**：《十日终焉》第511-512章（五人投票决定是否跟随齐夏）、第517章（「谁敢去」的抉择）
> **提炼方法**：从叙事与节奏、角色与心理、人性试炼维度中提取可复用技法
> **适用场景**：团队内部出现重大分歧需要决策时；需要制造团队内部分歧张力时
> **与 `trust-credit-transfer` 的区别**：trust-credit-transfer 聚焦信任的信用担保转移；本Skill聚焦决策过程的悬念设计——如何让投票本身成为叙事张力

---

## 技法1：观点三角法 (Opinion Triangle)

### 1.1 核心原则

三个角色分别代表三种立场（实用/信仰/理性），第四人成为关键摇摆票。读者会自然代入某个立场，而关键票的归属成为悬念核心。

### 1.2 设计步骤

1. **确定三种立场**：必须是三种不同但各有合理性的世界观/方法论
   - 实用主义：「跟随最有可能成功的人」（林檎）
   - 信仰主义：「追随最强大的力量」（云瑶）
   - 理性主义：「信息不足不决策」（章晨泽）
2. **让每种立场有充分论证**：不能有一种明显「错误」——读者必须觉得每种选择都合理
3. **设计第四人的摇摆**：第四人的选择理由必须出人意料——不是「被说服」，而是基于完全不同的动机
4. **关键票的理由**：甜甜选择云瑶的理由不是「认同计划」，而是「你对我好」——感恩比理性更真实

### 1.3 示例

```
林檎：「我们应该跟随齐夏，他是最有可能成功的」（实用主义）
云瑶：「许流年展现了神迹，我们应该追随更强的力量」（信仰主义）
章晨泽：「信息不全不做选择」（理性主义）
甜甜：「你对我好，你做的事就是我想做的事」（感恩驱动——出人意料的理由）
```

### 1.4 注意事项

- 三种立场不能有明显的道德高低——否则投票没有悬念
- 关键票的理由必须是「意料之外情理之中」——读者回忆前文后会恍然大悟
- 投票结果不能解决分歧——反而要让分歧更深（林檎意识到这意味着要「欺骗齐夏」）

## 技法2：投票悬念法 (Voting Suspense)

### 2.1 核心原则

将团队决策设计为逐人表态的投票过程，每人态度不同，关键票落在意想不到的人身上。

### 2.2 设计步骤

1. **逐人表态**：不要一次性宣布结果，而是让每个人依次表态
2. **态度梯度**：从明确支持→明确反对→弃权→出人意料的选择，形成梯度
3. **弃权者的功能**：弃权不是「不参与」，而是一种立场表达（李香玲的「小透明」定位）
4. **关键票延迟**：让最重要的票最后揭晓，前面的票制造期待
5. **结果反转**：最终结果与读者预期相反（预设甜甜会弃权，结果她明确站队）

### 2.3 注意事项

- 投票过程本身就是叙事——不要跳过过程直接给结果
- 每个人的表态都要揭示其性格/经历/价值观
- 投票结束后要有「后果反思」——投票结果带来的新问题比旧问题更严重

## 技法3：「谁敢去」抉择 (Who Dares to Go Choice)

### 3.1 核心原则

用「是否愿意暴露内心秘密」作为团队任务的筛选条件，通过每个角色的拒绝/接受暴露其人生经历和心理防线。

### 3.2 设计步骤

1. **设置高风险任务**：任务的代价不是物理伤害，而是心理暴露（被读心者看穿内心）
2. **逐人反应**：每个角色的拒绝/接受都揭示其人生经历
   - 李警官拒绝：「犯过不可饶恕的错」——暗示隐藏的过去
   - 乔家劲接受：「一生光明磊落」——坦荡的勇气
   - 陈俊南半推半就：「五毒俱全」——嘴硬心软
3. **拒绝比接受更有力**：「我不去」比「我去」更能揭示角色的内心

### 3.3 注意事项

- 「谁都不想去」比「谁都想去」更真实——拒绝才是人性的本能反应
- 拒绝的理由必须各不相同——反映不同的人生经历
- 最终的「去」必须是冲动而非理性——陈俊南「脑子一热还是站起身来」

## 验证清单

- [ ] 三种立场各有合理性，没有明显道德高低
- [ ] 关键票的理由是「意料之外情理之中」
- [ ] 投票过程是叙事的一部分，而非跳过的过程
- [ ] 每个人的表态都揭示其性格/经历
- [ ] 投票结果制造了新的问题（而非解决旧问题）
