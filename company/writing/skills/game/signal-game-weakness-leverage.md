# 信号博弈式卖破绽 Skill

> **核心来源**：《十日终焉》第548章——楚天秋教许流年：「既然你无论如何都会露出破绽，不如主动卖个破绽给他」。许流年想掩饰身份，楚天秋却建议她「给齐夏一个下马威」，让齐夏因看不起「楚天秋」而轻敌。策略的核心是：**承认弱点的存在，但将弱点转化为武器**。
> **提炼方法**：从策略与博弈、智斗与反转维度中提取可复用叙事技法
> **适用场景**：信息不对等的博弈、角色处于弱势时的策略翻转、对手有「看不起弱者」性格特征时

---

## 技法原理

博弈论中的「信号博弈」（Signaling Game）：发送者（卖破绽者）有私有信息，接收者（对手）根据发送者的信号来推断。关键洞察是：

1. **掩饰弱点的失败概率极高**——当对手足够聪明时，你无法长期隐藏弱点
2. **主动暴露弱点可以控制叙事**——你决定暴露哪个弱点、暴露多少、何时暴露
3. **暴露的弱点必须在可控范围内**——你暴露的弱点不能是致命的，必须是「可以被对手利用但不会导致你失败」的
4. **对手必须有「看不起弱者」的性格特征**——如果对手是谨慎型的，卖破绽反而会让他更警惕

核心公式：**弱点从「致命缺陷」变为「可控武器」**

---

## 适用场景

### 场景A：身份暴露的不可避免
当角色的身份/弱点注定会被发现时（如许流年的第三方身份），与其掩饰不如主动暴露。适用于：
- 间谍/卧底叙事
- 身份悬疑场景
- 信息不对等的博弈

### 场景B：以弱示强
当角色明显弱于对手时，主动展示弱点让对手低估自己。适用于：
- 以弱敌强的叙事
- 策略翻转场景
- 权力博弈

### 场景C：心理战
通过卖破绽来操控对手的判断和行为。适用于：
- 智斗场景
- 预判对手的预判
- 多层博弈

---

## 实现步骤

1. **识别弱点**：确定角色注定会暴露的弱点（身份、能力、意图等）
2. **评估对手**：确认对手是否有「看不起弱者」的性格特征
3. **设计暴露方式**：选择暴露哪个弱点、暴露多少、何时暴露
4. **设置陷阱**：在暴露的弱点中埋下「反利用」的伏笔
5. **收割结果**：当对手因低估你而犯错时，执行真正的计划

---

## ⚠️ 常见陷阱

### 陷阱1：暴露了致命弱点
卖破绽的前提是「暴露的弱点在可控范围内」。如果暴露的弱点是致命的（如暴露真实身份会导致立即死亡），策略就不成立。

### 陷阱2：对手不是「轻敌型」
如果对手是谨慎型/多疑型的，卖破绽反而会让他更警惕。这个策略只对「看不起弱者」的对手有效。

### 陷阱3：暴露后无法控制叙事
一旦暴露了弱点，你就失去了对叙事的控制。如果对手不按你预期的方式利用这个弱点，策略就会失败。

## ✅ 验证清单

- [ ] 弱点是否注定会被发现（掩饰失败概率极高）？
- [ ] 对手是否有「看不起弱者」的性格特征？
- [ ] 暴露的弱点是否在可控范围内（不是致命的）？
- [ ] 是否在暴露的弱点中埋下了「反利用」的伏笔？
- [ ] 是否有备选方案（如果对手不按预期利用弱点）？
