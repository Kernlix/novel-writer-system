# 🎣 黄金三章/钩子 Skill

## 命令
`/novel:hook`

## 功能
黄金三章和钩子设计。详细钩子类型和技法见 chapter-writing.md 中的"开篇钩子设计"和"黄金三章原则"章节。

## 高级开篇钩子模式

### 1. 密室/异常场景开场
主角在未知环境中醒来，与读者一样对情况一无所知：
- 封闭空间 + 陌生面孔 + 诡异规则
- 开场即建立强烈悬念："我是谁？这是哪？发生了什么？"
- 通过角色对环境细节的观察逐步揭示世界观

### 2. 规则即时展示
开篇同时展示核心规则：
- 规则通过角色对话自然展示，非旁白说明
- 规则的严苛性立即制造紧张感
- 规则的漏洞为后续反转埋下伏笔

### 3. 震撼性事件引爆
前3章内必须有一个震撼性事件：
- 不是简单的冲突，而是颠覆认知的事件
- 通过该事件确立**后果是真实的**
- 让读者和角色同时意识到：这不是开玩笑

### 4. 多角色速写
快速建立多个角色的第一印象：
- 每个角色用 2-3 个特征/动作/台词立住人设
- 不需要详细介绍，留白让读者自己判断
- 后续逐步深化和反转第一印象

## 平台适配
- **起点**：节奏偏快，金手指早期展现
- **番茄**：节奏极快，每章都有爽点/钩子
- **晋江**：情感铺垫为主，人物先行
