# 写手自查清单速查

> 动笔前和写完后各过一遍。

## 写作前

- [ ] 调用了知识检索（`--caller writer`）
- [ ] 参考了角色设计师和剧情架构师的同步产出
- [ ] 按模板1（基础续写）组装了提示词

## 写作中

- 对话质量 → 见 `knowledge/rules/common/dialogue-quality.md`
- 场景沉浸 → 见 `knowledge/rules/common/scene-immersion.md`
- 权力关系 → 见 `knowledge/rules/common/power-relationship.md`
- 字数：不低于2000字（仅计中文，不含标点空格数字英文）

## 写作后

- [ ] 标题准确反映本章内容（5-15字）
- [ ] 末尾不加"（本章完）"
- [ ] 系统内部术语未出现在外部对话中
- [ ] 自我检查基础质量标准

## 字数参考（非强制）

| 块类型 | 字数参考 |
|:------|:--------|
| 环境/氛围描写 | 100～200字 |
| 单人对话（一来一回） | 80～150字 |
| 内心独白/分析 | 150～300字 |
| 动作场面（每回合） | 200～400字 |
| 新信息揭示 | 200～400字 |
