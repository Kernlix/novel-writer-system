# epub提取指南

> epub文本提取的注意事项和最佳实践。

## 提取工具选择

| 方式 | 适用场景 | 缺点 |
|:----|:---------|:-----|
| `ebooklib` (Python) | 编程批量提取，保留结构 | 需安装依赖 |
| `calibre ebook-convert` | 命令行快速转换 | 输出格式不够精细 |
| 手动复制 | 单章快速查看 | 无法批量 |

## 输出格式建议

```
extracted/<卷号>/
├── 01-序章.txt
├── 02-第一章·和真死了.txt
├── 03-第二章·与女神相伴的日子.txt
└── ...
```

## 清洗规则

1. **插图/标签**：删除 `img/figure/figcaption` 标签
2. **空行**：合并3个以上连续换行为2个
3. **版权页**：跳过少于100字的片段
4. **特殊字符**：保留日文和中文标点，统一换行符

## 分章方式

epub内部通常以 `<h1>/<h2>` 或特定CSS类标记章节。提取时：

1. 先尝试匹配 `第X章` 标题
2. 若无标题，以文件分割为单位（每epub内部item = 一章）

## 性能

- 小文件（<10MB）：直接全量读取
- 大文件（>20MB，如图片多的epub）：先提取文本item列表，跳过图片
