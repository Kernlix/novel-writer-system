# ⏰ 时间线设计 Skill

> 适用于：需要为小说建立完整时间框架、检测时间矛盾、管理多故事线时间同步的场景。

## 核心技法

### 1. 时间锚点设置法

**公式**：`时间锚点 = 时间标记 + 事件 + 角色 + 位置`

| 锚点类型 | 说明 | 示例 |
|:---------|:-----|:-----|
| **硬锚点** | 不可移动的关键事件 | 大战、死亡、告白、离别 |
| **软锚点** | 可微调的过渡事件 | 日常、训练、旅行 |
| **隐锚点** | 未明说但读者能推断的时间 | "三天后"、"入秋时" |

**工作流程**：
1. 先确定所有硬锚点（故事骨架）
2. 在硬锚点之间填充软锚点（血肉）
3. 用隐锚点连接（呼吸感）

### 2. 时间矛盾检测法

**四类常见矛盾**：

| 矛盾类型 | 描述 | 检测方法 |
|:---------|:-----|:---------|
| **时间重叠** | 同一角色同时出现在两个地点 | 逐章检查角色位置 |
| **时间空白** | 关键事件缺少时间标记 | 检查事件间的时间跨度 |
| **时间膨胀** | 时间跨度与事件密度不匹配 | 对比时间长度与事件数量 |
| **时间压缩** | 大量事件挤在短时间内 | 检查单日事件数量 |

**检测清单**：
- [ ] 每章开头有时间标记吗？
- [ ] 角色位置清楚吗？
- [ ] 事件间的时间跨度合理吗？
- [ ] 多故事线时间同步吗？

### 3. 多线程时间管理法

**公式**：`时间同步 = 统一时间基准 + 并行事件对照`

**三线并行结构**：
```
主线A: 事件1 → 事件2 → 事件3 → 高潮
       ↓        ↓        ↓
支线B: 事件1 → 事件2 → 事件3 → 汇合
       ↓        ↓        ↓
暗线C: 事件1 → 事件2 → 事件3 → 揭露
```

**同步规则**：
- 每3章至少一次三线交叉点
- 交叉点必须有信息交换（不能只是"同时发生"）
- 暗线的时间标记要模糊（保持神秘感）

### 4. 时间节奏控制法

**公式**：`时间节奏 = 时间跨度 ÷ 事件密度`

| 节奏类型 | 时间跨度 | 事件密度 | 适用场景 |
|:---------|:---------|:---------|:---------|
| **慢镜头** | 短（秒/分） | 高 | 高潮、战斗、告白 |
| **正常** | 中（小时/天） | 中 | 日常、推进 |
| **快进** | 长（周/月/年） | 低 | 过渡、成长 |
| **跳跃** | 极长（年） | 极低 | 系列间隔、时代变迁 |

**节奏变化规则**：
- 高潮前：时间逐渐变慢（紧张感）
- 高潮后：时间逐渐加快（释放感）
- 日常段：时间均匀（舒适感）

### 5. 时间标记系统

**统一时间格式**：
```
第X天 | 上午/下午/晚上 | 事件描述
第X周 | 周一~周日 | 事件描述
第X月 | 上旬/中旬/下旬 | 事件描述
第X年 | 春/夏/秋/冬 | 事件描述
```

**时间标记位置**：
- 每章开头：`【第X天-上午】`
- 场景切换：`--- 三天后 ---`
- 闪回开始：`【回忆·第X天】`
- 闪回结束：`【回到现在】`

## 工作流程

1. **建立时间框架**：确定故事的时间跨度（天/周/月/年）
2. **设置硬锚点**：标记不可移动的关键事件
3. **填充软锚点**：在硬锚点之间添加过渡事件
4. **绘制时间线**：将所有事件按时间排序
5. **矛盾检测**：检查时间重叠、空白、膨胀、压缩
6. **节奏优化**：根据情感需要调整时间节奏
7. **输出时间线图表**：可视化时间结构

## 示例

### 简单时间线（单故事线）

| 时间 | 事件 | 角色状态 |
|:-----|:-----|:---------|
| 第1天 | 主角醒来，发现异世界 | 迷茫、恐惧 |
| 第3天 | 遇到导师，开始学习 | 希望、努力 |
| 第30天 | 完成基础训练 | 自信、成长 |
| 第90天 | 第一次战斗 | 紧张、突破 |
| 第180天 | 面对强敌 | 坚定、成熟 |

### 多线程时间线

| 时间 | 主线A | 支线B | 暗线C |
|:-----|:------|:------|:------|
| 第1天 | 主角出发 | 反派密谋 | 神秘人观察 |
| 第3天 | 到达城镇 | 反派布局 | 神秘人留下线索 |
| 第7天 | 结识伙伴 | 反派动手 | 神秘人身份暗示 |
| 第14天 | 第一次危机 | 反派暴露 | 神秘人真实身份 |

## 常见错误

| 错误 | 后果 | 正确做法 |
|:-----|:-----|:---------|
| 时间标记不统一 | 读者混乱 | 全书使用统一格式 |
| 时间重叠 | 逻辑漏洞 | 逐章检查角色位置 |
| 时间空白 | 悬念断裂 | 补充过渡事件 |
| 时间膨胀 | 节奏拖沓 | 压缩或扩展时间跨度 |
| 多线不同步 | 混乱 | 建立统一时间基准 |
