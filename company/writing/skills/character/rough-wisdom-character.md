# 莽夫面具下的心机角色设计技法

> 「妈的……还不下班？」——表面在抱怨加班，实际在等人。
> 「谁走我就告发谁，妈的你俩谁也跑不了。」——把"商量"变成"既成事实"。——地虎·第398章

## 概述

「莽夫面具下的心机」是反直觉角色设计中最有效的一种。角色的**表面**（粗鲁、暴躁、直率）是社交面具，**内核**（算计、预判、布局）是真实能力。读者和一个莽夫相处了四分之三的篇幅，最后发现他一直在下棋。

这种设计的威力在于：**最不需要防备的人，往往最危险**。读者和角色都放松警惕时，莽夫的突然"聪明"带来的冲击力远大于一个本就被设定为聪明的人。

---

## 技法结构

### 核心公式

```
莽夫面具（粗鲁语言 + 暴脾气 + 冲动形象）
  +
心机内核（精准计算 + 预判布局 + 操控他人）
  =
持续惊喜的角色
```

### 设计三原则

| 原则 | 说明 | 示例（地虎） |
|:-----|:------|:-------------|
| **语言粗糙** | 对话充满粗口、抱怨、不耐烦的语气 | 「你他妈扯什么犊子」「妈的」 |
| **行为有目的** | 每个看似随意的行为背后都有明确意图 | 抱怨加班→实际在等人 |
| **关键时刻揭底** | 在需要决策时展示真正的计算能力 | 「拉所有知情者入伙」降低告密风险 |

### 执行步骤

1. **建立粗鲁形象**：前30%的出场时间只展示粗鲁的一面——骂人、抱怨、冲动
2. **植入可疑的"巧合"**：让敏锐的读者开始怀疑——"他做的每件事好像都有目的"
3. **小规模展示心机**：在次级博弈中展示计算能力——"原来他刚才那句话是故意的"
4. **关键时刻的大揭示**：在需要站队的核心博弈中，展示全套计算链

## 应用场景

| 场景类型 | 适用度 | 说明 |
|:---------|:------|:------|
| 造反/叛变情节 | ⭐⭐⭐⭐⭐ | 莽夫是造反的最佳策划者（因为没人防备他） |
| 底层/市井角色 | ⭐⭐⭐⭐⭐ | 底层生存环境培养出来的"粗鲁的外交家" |
| 雇佣兵/杀手角色 | ⭐⭐⭐⭐ | 最危险的杀手往往看起来最无害 |
| 搞笑+智斗混合 | ⭐⭐⭐⭐ | 喜剧的外壳包裹智斗的内核 |
| 群像中的差异化 | ⭐⭐⭐⭐ | 在一群"明显聪明"的角色中需要一个"隐藏聪明"的人 |

## 配合指南

### 前置条件
- 角色必须有一种"让人不想和他多说话"的气场
- 角色必须有至少一次"粗鲁但没有造成损失"的正面案例（建立信任）
- 需要有一个更聪明的角色来"识别"莽夫的真正水平（如地羊对地虎的了解）

### 避坑指南
| 陷阱 | 说明 |
|:-----|:------|
| **太早揭底** | 至少要在读者接受了"他就是个莽夫"的印象后再揭示——过早会破坏反差 |
| **全天候心机** | 合理的设计是：平时是莽夫，关键决策时才切换为心机模式 |
| **计算过于精确** | 莽夫的"心机"应该是直觉式的（如"我知道拉他入伙比赶他走安全"），不是公式推演 |
| **真愚蠢 vs 装愚蠢** | 要区分"真蠢但偶尔聪明"和"装蠢但真的很聪明"——前者是人设崩坏，后者是角色深度 |
| **无代价的心机** | 莽夫的粗鲁必须有时给他带来麻烦（如得罪了不该得罪的人），否则就太完美了 |
