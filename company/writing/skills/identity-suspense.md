# 身份悬疑 Skill

## 技法A：多重误认结构

同一秘密身份/物品被不同角色赋予完全不同的含义，互不交叉但层层叠加：

```yaml
物品: 义贼盗取的神器戒指
视角层:
  和真: 贼赃，不能被发现，想办法处理掉
  达克尼斯: 国家机密，涉及王室安全
  爱丽丝: 婚约信物，个人情感象征
  阿库娅: 装饰品，好看但没用
  克里斯: 任务目标，必须回收
```

每层都是独立的"事实"而非"误解"——因为戒指确实同时是以上所有东西。

## 技法B：揭露时机四阶段

```
近泄露 → 角色差点暴露身份 → 紧张建立
半揭露 → 部分人知道，部分人不知 → 信息差张力
伪揭露 → 以为被发现了（其实对方在说另一件事）→ 喜剧缓冲
真揭露 → 身份真相大白 → 情感最高点
```

## 技法C：信息防火墙

角色知情权分层表——每次跨层传递都是一个剧情节点：

| 角色 | 知情等级 | 知道什么 | 不知道什么 |
|:-----|:-------:|:---------|:----------|
| 克里斯 | 全知 | 义贼身份+神器目的+和真的盗窃 | — |
| 和真 | 3级 | 义贼=克里斯，戒指=赃物 | 戒指=国家机密=婚约信物 |
| 同伴 | 0级 | 无 | 全部 |
