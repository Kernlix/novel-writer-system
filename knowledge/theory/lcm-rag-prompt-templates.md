# LCM + RAG 写作提示词模板库

> 适配DeepSeek 1M超长上下文 + 双层RAG检索，长短篇通用。
> 所有模板中 `{rag内容}` 部分由知识检索智能体在调用时动态填充。

---

## 一、上下文填充顺序（固定结构）

所有模板统一遵循以下信息排列顺序：

```
1. 【RAG检索参考资料】 ← 最上方，模型先读取设定
2. 【近期正文片段】    ← 中间，模型理解当前剧情位置
3. 【写作需求/规则】   ← 末尾，模型对末尾指令记忆更牢
```

硬性规则（所有模板自动附加）：
> **参考资料无记载的信息一律不写，如有信息缺失直接标注，禁止自行补充设定。**

---

## 二、五个通用模板

### 模板1：基础续写（写手日常写作最高频）

```markdown
# 基础写作约束
你拥有超长文本读取能力，严格遵循所有提供的参考资料，禁止凭空编造设定、人物、剧情、道具。

【检索参考资料】
{rag召回内容：人物卡/世界观/前文伏笔/道具/势力设定}

【近期正文片段】
{最近1~5章原文，根据需求增减长度}

写作需求：{剧情指令：续写本章/推进支线/写对峙桥段/过渡日常}

硬性规则：
1. 人物行为、心理、对话完全贴合档案人设，杜绝OOC；
2. 主动呼应前文所有相关伏笔、信物、恩怨、过往剧情；
3. 严格遵守世界体系规则（修炼、时代、国家、律法、种族特性），不新增未定义设定；
4. 文风、句式节奏、叙事视角和前文保持统一；
5. 剧情逻辑连贯，支线不无故断掉，配角行为符合自身立场；
6. 无冗余灌水文字，情节有推进，细节符合故事氛围；
7. 若资料存在矛盾，优先沿用更早前文内容，并标注冲突点。
```

### 模板2：复盘查漏（审查官自查/全卷校验）

```markdown
# 全文校验任务
你具备百万字长文本通读能力，仔细审阅下方全部资料与正文，排查所有逻辑漏洞。

【完整设定库】
{rag检索：全部人物、世界规则、历史时间线、道具记录}

【目标审阅文本】
{单章/整卷/全部已写内容}

需要完成：
1. 人物一致性检查：核对所有人言行是否匹配人物档案，标记人设崩坏段落；
2. 伏笔校验：记录前文埋下、后文无回收的伏笔，以及前后矛盾的道具/地名/时间；
3. 时间线梳理：标出时间错乱、年份冲突、事件先后颠倒的内容；
4. 势力逻辑检查：各国/宗门/家族行为是否符合自身利益设定；
5. 破折号审计：全文每章破折号是否≤3处，且仅用于声音延长/话语中断；
6. 对话质量：是否存在冗余问答、角色对话辨识度是否足够、潜台词是否到位；
7. 场景沉浸：感官描写/时间痕迹/空间关系是否交代清楚；
8. 权力关系：领主与下属的权力基调是否一致，沉默是否被误写成畏惧；
9. 情感变化：本章情感基调是否与前后章形成变化，刀子情节后是否有缓冲；
10. 剧情节奏：反转/悬念/钩子使用是否合理，高潮前是否有缓冲；
11. 语言质量：无语病/无AI套话/无口语赘词；
12. 输出格式：输出12项gates（basic/dash_audit/narrative/pacing/character/plot/language/dialogue/consistency/immersion/power_relation/emotion），先汇总全部问题清单，再分段给出修改方案。
```

### 模板3：大纲推演（剧情架构师规划）

```markdown
# 剧情规划任务
基于完整世界观与已写内容，推演合理剧情走向，保证长线逻辑自洽。

【基础设定】
{rag：大陆格局、人物关系、核心冲突、核心伏笔、故事核心主题}

【现有已写剧情】
{前文全部关键剧情节点}

【规划要求】
{需求：写全卷大纲/设计支线/设计中期高潮/构思结局/安排配角归宿}

约束：
1. 所有剧情走向不能违背人物底层性格与势力立场；
2. 尽可能回收现有伏笔，不强行新增大量无关设定；
3. 主线、支线穿插节奏均衡，避免单一角色长期独占剧情；
4. 分阶段划分清晰：铺垫→冲突升级→高潮→回落收尾；
5. 标注每个阶段关键信物、关键对手、关键转折事件。
```

### 模板4：改写润色（润色师/文风调整）

```markdown
# 文本润色改写任务

【参考基准资料】
{人物设定、文风参考段落、世界观氛围要求}

【待修改原文】
{需要调整的章节/段落}

修改需求：{扩写细节/精简冗余/压抑伤感文风/热血打斗/第一人称转第三人称/删除灌水/优化对话自然度}

硬性要求：
1. 核心剧情、关键对话、重要伏笔完整保留，不得删减关键信息；
2. 全程贴合人物性格，修改后不出现人设偏移；
3. 统一全文语感，消除段落间文风割裂；
4. 按需增加环境、心理、动作细节，或压缩无效描写；
5. 打斗/情感场面节奏适配故事整体基调。
```

### 模板5：独立人物/支线（角色设计师专项）

```markdown
# 人物支线创作

【全局世界观】
{rag召回世界基础规则、时代背景、势力划分}

【目标人物完整档案】
{完整人物卡：出身、性格、执念、弱点、人际关系、隐藏秘密、过往创伤}

【关联前文剧情】
{该角色登场过的所有章节片段、相关恩怨伏笔}

创作需求：{写人物番外/单独支线/角色个人独白/该人物专属冲突桥段}

规则：
1. 所有行为贴合人物底层执念与创伤，行为逻辑自洽；
2. 联动该角色与其他角色的过往交集，呼应相关伏笔；
3. 番外/支线不破坏主线既定剧情走向；
4. 突出人物独有特质，避免角色同质化。
```

## 三、五大高频配套提示词

### 功能1：批量整理人物关系&时间线

```markdown
通读下方全部设定与章节文本，整理两份清晰表格：
1. 全人物关系表：包含身份、阵营、爱恨、恩怨、隐藏羁绊；
2. 完整时间线：按时间顺序记录所有重大事件、相遇、冲突、伏笔节点；

资料：{rag全部人物档案+所有关键章节原文}

输出要求：条理清晰，分段排版，标注每个事件对应的章节出处，标记未回收伏笔。
```

### 功能2：伏笔回收设计

```markdown
读取全部前文，提取所有伏笔、信物、秘密、预言、未了结恩怨，
结合当前剧情阶段，设计自然的回收方案：
1. 列出所有未回收伏笔清单；
2. 分配到后续章节，标注适合触发的场景、对应人物；
3. 设计不突兀、符合人物逻辑的回收桥段；

资料：{前文全文+人物设定+道具档案}
```

### 功能3：桥段生成（打斗/对峙/告白/权谋）

```markdown
基于提供的人物、势力、前文恩怨，创作指定桥段：
桥段类型：{对峙/宗门大战/朝堂谈判/情感告白/生死诀别/密室密谋}
氛围要求：{压抑/热血/悲情/诡谲/温情}

硬性限制：
1. 对话贴合双方人物性格；
2. 冲突根源来自前文已有的矛盾，不强行制造新冲突；
3. 桥段推进主线或支线，不写无意义独立剧情；

参考资料：{相关人物卡、过往交集片段、世界观战力规则}
```

### 功能4：新书开篇/卷首开篇生成

```markdown
创作小说开篇，要求快速建立世界观、抛出核心冲突、塑造主角形象、留下阅读钩子。

参考资料：
1. 世界基础设定：{时代、大陆、力量体系、核心矛盾}
2. 主角完整人物档案：{性格、困境、执念、目标}
3. 前期核心伏笔：{关键信物、宿命、仇家}

约束：开篇不堆砌大段设定，用剧情自然带出背景，结尾留下悬念。
```

### 功能5：全文降重/去灌水

```markdown
通读全文，精简冗余描写、重复心理、无效对话、流水账日常；
保留全部核心剧情、伏笔、关键对话、人物高光段落；
精简后保持原有文风、人设、剧情走向不变；
标出大幅删减的段落，并给出优化后的完整正文。

待处理文本：{章节原文}
参考设定：{人物、世界观，修改时不能违背设定}
```

## 四、模板选用速查

| 场景 | 使用模板 | 调用方 |
|:----|:--------|:-------|
| 日常写新章节 | **模板1** 基础续写 | 写手Agent |
| 写完后自查一致性 | **模板2** 复盘查漏 | 审查官Agent |
| 规划全卷/支线 | **模板3** 大纲推演 | 剧情架构师Agent |
| 修文/改文风/扩写精简 | **模板4** 改写润色 | 润色师Agent |
| 新增角色/写番外 | **模板5** 独立人物/支线 | 角色设计师Agent |
| 整理关系表时间线 | **功能1** | 剧情架构师Agent |
| 排查未回收伏笔 | **功能2** | 审查官/剧情架构师 |
| 写特定桥段 | **功能3** | 写手Agent |
| 开篇/卷首 | **功能4** | 写手Agent |
| 降重去灌水 | **功能5** | 润色师/审查官 |
