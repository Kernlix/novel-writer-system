# 角色语言高度差异化——不看名知人

## 原文/原片段

**和真（吐槽役）：**
> 「喂喂喂，这跟说好的不一样吧？」
> 「吵死了！你除了哭还会什么！」
> 「你这也配叫女神？连个青蛙都打不过的神算什么神？」

**阿库娅（傲慢哭包）：**
> 「呜哇——！和真你这个废柴！都是你的错！」
> 「哼！我可是女神！区区青蛙……区区青蛙……快跑啊和真！！」
> 「你说什么？！我可是水之女神阿库娅大人哦！你居然敢用这种态度跟我说话！」

**惠惠（中二爆裂狂）：**
> 「我名为惠惠！乃操纵爆裂魔法之人！」
> 「Explosion！！！」
> 「不……那个云它实在太黑了，简直就像在嘲笑我……」

**达克尼斯（变态抖M骑士）：**
> 「你这个下贱的男人……！居然……居然想让我做这种事……！」
> 「不……不要管我！让我就这样被怪物蹂躏至死……！」
> 「呵……你也就趁现在还能威风了……（表情愉悦）」

## 技法分析

素晴的角色语言差异已经到了"不用看角色名都知道是谁"的程度。这并非巧合，而是**刻意设计的语音签名（Voice Signature）**：

每个角色的语言都有三层差异化：

1. **词汇层特有的专属用词**
   - 惠惠：Explosion、我名为XXX、爆裂魔法
   - 阿库娅：废物和真、我可是女神、呜哇
   - 达克尼斯：下贱的、蹂躏、不要管我

2. **句法层特有的说话节奏**
   - 惠惠：长短交错的中二式演讲节奏
   - 阿库娅：短句+哭腔+喊叫
   - 达克尼斯：欲拒还迎的断句，伴随省略号

3. **语用层特有的语言动机**
   - 和真：每句话都在吐槽/反驳
   - 阿库娅：每句话都在甩锅/自夸
   - 惠惠：每句话都在给爆裂魔法找理由
   - 达克尼斯：每句话都在往被虐方向带

**核心原理：角色的语言不是"这个人会说的话"，而是"只有这个人才会说的话"。**

## 可复用场景

- 多主角/群像剧（必须使用，否则读者会混淆）
- 任何有3个以上主要角色的作品
- 需要快速建立角色辨识度的开篇章节

## 自我提升方案

### 对照分析

灵境当前的"角色反应模板化"检测（第8条）只能识别角色"做什么"的模板化，但无法识别角色"说什么"的同质化。

目前润色策略中的"口语化对话"只是一个笼统的方向，缺乏**逐角色差异化的具体方法**。一个常见的缺陷是：润色后的对话虽然口语化了，但所有角色说的都是"同一种口语"——大家都像同一个人在说话。

### 优化措施

1. 在ai-detection-signals中新增"角色语言同质化"检测维度
2. 设计"语音签名"模板用于润色前准备：

   ```
   角色语音签名表
   ├─ XXX角色
   │  ├─ 专属词汇：[词1, 词2, ...]
   │  ├─ 口癖：[固定起手式/尾缀]
   │  ├─ 句式偏好：[短句型/长句型/反问型/感叹型/省略型]
   │  ├─ 语气词：[啊/呀/呢/哦/哼/呵]
   │  └─ 骂人风格：[直球/阴阳怪气/一本正经]
   ```

3. 在润色流程中增加"去标签测试"：在润色完成后，随机选取5段对话去掉说话人标签，由润色师判断是否能正确归位

### 应用承诺

在每次接手润色任务前，先阅读角色档案，为每个主要角色建立语音签名。润色过程中的对话部分，逐句检查是否命中该角色的语音签名特征。未命中的句子标记为"角色语言漂移"，修改至命中为止。
