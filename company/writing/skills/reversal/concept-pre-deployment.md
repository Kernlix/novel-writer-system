# 概念提前投放（Concept Pre-deployment）

> 在不解释的情况下提前投放未定义的超自然概念/术语，制造悬疑引导读者继续阅读。

## 定义

在角色对话中首次出现一个未定义、未解释的神秘概念或术语，不展开说明其含义，让读者带着疑问继续阅读。这是一种**悬疑式信息投放**手法——先埋名再解释。

## 适用场景

- 超自然/奇幻/科幻题材中引入新设定时
- 需要在早期制造长效悬念时
- 世界观体系复杂，需要分阶段揭示时

## 技法要点

### 1. 投放时机

- **早期投放**：在读者还没有形成完整世界观认知时投放，让术语成为读者的"待解之谜"
- **自然嵌入**：术语应通过日常对话自然带出，不刻意、不突兀
- **频率控制**：首次投放后，在适当间隔后再次提及或部分解释

### 2. 术语设计

- **名称简洁有力**：单字或两字最好（如「化形」「回响」「极道」）
- **暗示但不暴露**：术语本身暗示某种能力/存在/体系，但不明确具体是什么
- **反差感**：术语的含义应与读者的初步猜测不同

### 3. 对话植入

- **角色提问**：让一个不了解的角色提问"你是那个XX吗？"——既自然又制造疑问
- **不做定义**：被问者不正面回答，或者给出一个模糊否认/确认
- **遗留未解**：对话自然转到其他话题，留下未回答的疑问

### 4. 后续揭示

- **延迟满足**：不要在同一章或紧接着的章节解释，让读者带着疑问阅读多章
- **分层揭示**：第一次解释给出部分信息，第二次解释深入，第三次完全揭示
- **反转预期**：最终揭示的含义应颠覆读者基于名称的初期猜测

## 示例对比

### ❌ 错误做法
```
"你会「化形」吗？"
"是的，我可以变成别人的样子，这是一种变形能力。"
```
直接解释，破坏悬念。

### ✅ 正确做法
```
"你是那个「化形」吗？"
齐夏没有回答，只是沉默地看着对方。
"算了，你不想说就算了。"
```
不解释、不否认、不回答——让术语成为悬疑钩子。

## 与相关技法的区别

| 技法 | 核心 | 区别 |
|:-----|:-----|:------|
| 身份悬疑 | 隐藏角色真实身份 | 概念提前投放不涉及身份，只涉及未定义的术语/概念 |
| 传统伏笔 | 暗示将要发生的事件 | 概念提前投放投放的是"名词"而非"事件" |
| 真相层次揭露 | 分层揭示已发生事件的真相 | 概念提前投放是在事件发生前先埋概念名 |
| 预知悲剧叙事 | 先告知结局再讲述故事 | 概念提前投放先告知"有这个概念存在"而非结局 |

## 来源

- **作品**：《十日终焉》

- **示例分析**：在角色对话中出现未定义的超自然概念，是典型的悬疑信息投放手法
