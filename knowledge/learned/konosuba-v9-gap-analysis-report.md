## 二、新增Domain识别（vs V1-V8基线）

### 🔴 Domain 1: 相互治愈型告白/悲喜交织感情技法（Mutual-Healing Confession）

**判断**: ⚠️ **现有 `romance-writer` 的Sub-skill不覆盖此技法**

V5建立了 `romance-writer` Agent 及其三个Sub-skill：`romance-progression`（感情线渐进四阶段模型）、`romance-anti-climax`（反高潮告白技法）、`action-substitute-confession`（行动替代告白技法）。V8新增 `love-triangle-romance`（多角感情线并行）。

但V9ch5的告白场景引入了一种**全新模式**——不只是一方告白或反高潮，而是「**相互暴露脆弱→通过比烂黑历史→共同治愈**」的双向情感释放。晓枣将其称为「**不完美告白**」——惠惠的「喜欢清单」全是和真的缺点，和真的回应是用自嘲黑历史消解惠惠的自责。

| ID | 本能核心 | 来源 |
|:---|:--------|:----|
| `konosuba:v9ch5-emotion-01` | **相互治愈型告白**——惠惠告白后紧接两人互诉黑历史（惠惠杀恩人/和真对父母愧疚），形成共鸣式互相治愈而非单方安慰 | ch5 Agent 6 |
| `konosuba:v9ch5-emotion-02` | **「不完美清单」告白模式**——惠惠列举和真全部缺点作为喜欢理由（「不完美才是真物的魅力」），打破传统告白套路 | ch5 Agent 3 |
| `konosuba:v9ch5-comedy-03` | **三重情感转向技法**——紧张告白→喜剧打断(阿库娅破门)→吐槽回应(爆裂魔法)→再认真→最终释然；三次转向而不中断情感线 | ch5 Agent 7 |
| `konosuba:v9ch5-emotion-04` | **「比烂治愈」心理机制**——和真用「我的黑历史比你更糟」来消解惠惠的自责，而非安慰或说教；通过自降尊严实现互相治愈 | ch5 Agent 6 |
| `konosuba:v9ch5-char-05` | **告白后POV切换的结构意义**——第9章后尾声1切换至惠惠POV，标志着关系进入需要双视角呈现的新阶段 | epi Agent 1 |
| `konosuba:v9epi-emotion-06` | **「行动替代言语」的浪漫收束**——惠惠「决心留长头发」回应和真随口说的理想型（长直发），全卷最浪漫的动作式回应 | epi Agent 1 |

**≥3条**: ✅ **6条本能，远超阈值**

**与现有 `romance-writer` Sub-skill的差异**:

| Sub-skill | 核心模式 | V5/V8代表 | V9是否覆盖 |
|:----------|:--------|:----------|:----------|
| `romance-progression` | 握手→告白→公开确认四阶段 | V5的「被褥→我的男人」链 | ❌ 不覆盖双向治愈模式 |
| `romance-anti-climax` | 告白被喜剧打断 | V5的「告白→推销包装」 | ⚠️ 部分覆盖打断机制，但不覆盖「打断后重新真挚」的回环 |
| `action-substitute-confession` | 行动替代言语告白 | V5的「替按按钮」 | ❌ 完全不同 |
| `love-triangle-romance` | 多角感情线并行管理 | V8的烟花约会+颊吻 | ❌ 无关 |

**建议**: 
- ❌ **不建新Agent**——属于 `romance-writer` 领域内的进阶技法型
- ✅ **新增Sub-skill/模块**: `mutual-healing-confession`（相互治愈型告白技法）
  - **归属**: `romance-writer` Agent
  - **核心技法**:
    1. **相互脆弱暴露结构** — 双方各自暴露最深的黑历史/羞耻，形成共鸣而非单方安慰
    2. **「不完美清单」告白模式** — 通过列举对方的缺点来表白（反差萌的告白变体）
    3. **三重情感转向节奏** — 紧张→打断→吐槽→再真挚的告白节奏管理
    4. **「比烂治愈」表达心理学** — 用自嘲降低姿态来消解对方自责的独特安慰机制
    5. **行动式回应** — 告白后不求立即确认关系，而是通过单方面改变（如留长发）作为浪漫收束

## 三、最终研判：需新增Agent/Skill

### 核心结论：**不需新增Agent，需新增1个Skill + 1个Sub-skill扩展**

| 优先级 | Domain | 本能数 | 处理方式 |
|:---:|:---|:---:|:---|
| 🔴 **P0** | 相互治愈型告白技法 | **6** | **✅ `romance-writer` 新增Sub-skill: `mutual-healing-confession`** |
| 🔴 **P0** | 神格矮化学/神性解构 | **7** | **✅ 新增Skill: `godhood-dwarfing`（神格矮化学）** |
| 🟡 **P1** | 楔形叙事/双线反向揭示 | 4 | 吸收进 `plot-outline` Skill 扩展 |
| 🟢 **P2** | 告白后关系心理学 | 3 | 吸收进 `romance-writer` 的 `romance-progression` |
| 🟢 **P2** | 传送游击战 | 4 | 吸收进 `worldbuilding` |
| 🟢 **P2** | 卷末双向蓄力 | 4 | 吸收进 `plot-outline` |

### 为什么不需要新增Agent？

1. **相互治愈型告白**是 `romance-writer` 领域内的进阶技法变体——`romance-writer` 已覆盖反高潮、行动替代、三角关系管理，新增Sub-skill即可覆盖「双向脆弱暴露治愈」模式。

2. **神格矮化学**是 `humor-writer` 领域的扩展——神性解构本质是喜剧的世界观侧技法。虽然跨越了 `isekai-culture-clash` 和 `defect-comedy-engine` 的边界，但核心仍是「如何让神/高位存在变得好笑」，归属humor体系。

3. 已有4个Agent（humor / romance / identity-suspense / pseudo-family）覆盖素晴所有主要写作领域。新增1个Skill + 1个Sub-skill即可填补V9的技法空白。

## 六、V9四个重点关注问题逐条回答

### 1. V9ch5惠惠告白——是否需要升级 `romance-writer` 的「告白」Sub-skill？

**✅ 是。需要新增 `mutual-healing-confession` Sub-skill。**

V9ch5的告白不是V5「反高潮告白」或「行动替代告白」的重复——它首次引入了「双方同时暴露最深的黑历史→互相治愈→同时释然」的双向脆弱结构。现有的 `romance-anti-climax` 仅覆盖「告白被喜剧打断」，不如V8的「喜剧打断后重新真挚」回环，而V9将此模式推至极致。建议作为Sub-skill加入 `romance-writer`。

### 2. 沃尔巴克（邪神+前女神）的「神格矮化学」——是否已有覆盖？

**❌ 无充分覆盖。需要新增 `godhood-dwarfing` Skill。**

V8曾判断「女神日常化」已有 `isekai-culture-clash` 覆盖。但V9展示的沃尔巴克弧线远超出「日常化」范围——涉及神性结构解构（半身分离）、神格政治性（教团标签定义神格）、神的物理脆弱化（力量流失）、以及神的半身宠物化（猫化）。这些是 `isekai-culture-clash` 不覆盖的深层神性解构技法。

### 3. 逗之助=邪神半身的设定展开——是否需要新skill？

**建议纳入 `godhood-dwarfing` Skill 的「半身宠物化」子模块。**

逗之助的展开（宠物→半身→力量容器→潜在转生）是「神格矮化学」的自然延伸。不需要独立Skill——「神的半身被养为宠物」是 `godhood-dwarfing` 的核心技法之一（神的强势属性封印为可驯化生物）。建议作为该Skill的第5技法板块。

### 4. ≥3条同domain无覆盖→招募

## 八、V9独特贡献总结

第9卷对素晴写作体系的**三大贡献**：

1. **告白场景的终极进化** — 从V5的「反高潮傲娇告白」、V8的「喜剧打断→重新真挚」到V9的「相互脆弱暴露治愈」，《素晴》的告白技法完成了一次方法论的升级。这是晓枣最成熟的情感场景写作。

2. **神格矮化学的系统化** — 沃尔巴克展示了神性解构的完整光谱：从前女神→邪神→半身分离→猫化→宠物→转生钩子。这超越了 `isekai-culture-clash` 的「日常化」浅层，深入到神性结构本身的解构。

3. **双线索楔形叙事的成熟** — 反派消灭（线性向下的红色线索）+ 宠物揭示（反向追溯的黑色线索）在同一个交叉点相遇。这是 `plot-outline` 技法的进阶应用，值得作为独立模块归档。
