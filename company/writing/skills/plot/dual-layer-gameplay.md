# 双层博弈结构 Skill

核心来源：十日终焉第161-170章——兵器牌双层博弈
提炼方法：从死亡游戏机制、心理博弈、人性探讨相关维度中提取可复用叙事技法
适用场景：死亡游戏、策略博弈、团队协作、远程指挥、信息不对称类章节

---

## 技法1：双层博弈的叙事结构

将智力决策与体力执行分离，创造指挥官与执行者的叙事张力。

### 1.1 双层结构的定义

斗智层：策划者在方桌区，看不到搏斗者，核心能力是计算、推理、操控
斗勇层：搏斗者在玻璃房，看不到策划者的牌，核心能力是勇气、体力、应变

十日终焉案例：
- 齐夏在方桌前出牌，决定赵医生获得什么武器
- 赵医生在玻璃房内搏斗，只能凭掉落的道具判断齐夏出了什么牌
- 两个空间通过道具掉落连接，形成信息隔离

### 1.2 双层结构的叙事功能

1. 指挥官的无力感：齐夏的智谋再精妙，控制不了赵医生的恐惧
2. 执行者的被动性：赵医生不知道齐夏的策略，只能凭本能反应
3. 信息壁垒的焦虑：读者比任何单一角色都知道更多信息，产生全知视角的焦虑感
4. 远程协作的信号系统：策划者与搏斗者几乎无法沟通，只能用反常行为传递信息

### 1.3 双层结构的写作模板

```
斗智层：策划者在外部博弈
齐夏盯着苏闪的眼睛，手指在牌桌上轻轻敲击。
他在计算——赵医生能撑几个回合？

道具传递：两个空间的唯一连接
天花板的窗口掉落了一把绳子。
赵医生接住绳子，抬头看向玻璃墙外的齐夏。
齐夏微微点头。

斗勇层：搏斗者在内部执行
赵医生握紧绳子，面对子晨。
十秒倒计时开始。
他不知道齐夏为什么要给他绳子——但他选择相信。

信息壁垒：两个空间的焦虑
齐夏看到赵医生握紧绳子，嘴角微扬。
他知道赵医生会怎么做——但赵医生不知道。
这种信息差，是博弈的核心。
```

## 技法2：信息壁垒的设计与打破

信息壁垒是双层博弈的核心——如何制造信息差，又如何打破信息差。

### 2.1 信息壁垒的类型

物理壁垒：空间隔离（玻璃墙），制造看得见但够不着的焦虑
规则壁垒：游戏规则禁止沟通，强制信息不对称
时间壁垒：十秒行动窗口，消除反应时间差异
心理壁垒：角色间的不信任，增加协作难度

### 2.2 打破信息壁垒的方式

十日终焉案例：死牌——强制休息加策划者与搏斗者会面

1. 规则漏洞：利用死牌的特殊效果创造沟通窗口
2. 反常行为：赵医生的撒泼打滚实际是给齐夏传递信号
3. 默契配合：齐夏读懂赵医生的意味深长的眼神
4. 提前预判：齐夏在暂停期间就安排了赵医生的每一个动作

### 2.3 信息壁垒的写作模板

```
壁垒制造：物理或规则层面的信息隔离
玻璃墙将两人隔开——赵医生能看到齐夏，但听不到他说什么。
齐夏能看到赵医生在挨打，但无法阻止。

焦虑积累：信息缺失导致的紧张感
赵医生被子晨骑在身下暴打，齐夏却只能看着。
他不知道赵医生能不能撑住——信息差制造了焦虑。

信号传递：用反常行为打破壁垒
赵医生突然撒泼打滚，大喊我要刀子！
齐夏愣了一秒——然后嘴角上扬。
我就知道。

壁垒打破：信息传递成功
齐夏读懂了赵医生的信号——他的示弱是表演。
赵医生也读懂了齐夏的回应——他知道我是装的。
信息壁垒在这一刻被打破。
```

## 技法3：远程协作的信号系统

当直接沟通被禁止时，角色如何用编码后的信号传递信息？

### 3.1 信号类型

行为信号：用反常行为传递信息，可靠性中等
道具信号：用道具的使用方式传递信息，可靠性较高
表情信号：用微表情传递信息，可靠性较高但需要默契
时间信号：用行动时机传递信息，可靠性较低

### 3.2 信号的编码与解码

编码原则：
1. 反常即信号：正常行为是噪音，反常行为是信息
2. 重复即确认：一次可能是巧合，多次就是信号
3. 代价即诚意：付出代价的信号更可信

解码原则：
1. 观察行为模式：不是看单一行为，而是看行为序列
2. 排除干扰项：区分真实的崩溃和表演性的崩溃
3. 建立默契：需要长期合作才能建立信号解码能力

### 3.3 信号系统的写作模板

```
编码：角色用反常行为传递信息
赵医生突然躺在地上，大喊大叫。
子晨愣住了——这个人怎么突然疯了？
但赵医生的眼睛，偷偷瞄向玻璃墙外的齐夏。

解码：主角读懂了信号
齐夏看着赵医生的表演，嘴角微扬。
苦肉计……他低声说，你在骗她的盾牌。

确认：信号被确认
赵医生看到齐夏点头，继续他的表演。
他知道——齐夏读懂了。

代价：信号的代价
赵医生被打得口鼻流血——这是信号的代价。
但这个代价，换来了苏闪的盾牌。
```

## 技法4：策划者与搏斗者的信任构建

在信息壁垒和生死压力下，信任如何建立？

### 4.1 信任的三个阶段

被迫合作：被规则绑定，缺乏信任，制造初始张力
信号验证：通过信号传递建立初步信任，展示角色智慧
默契配合：不需言语的协作，展示团队成长

### 4.2 信任构建的写作原则

1. 信任必须被测试：不能直接建立信任，需要通过事件验证
2. 信任必须有代价：赵医生被打等于信任的代价
3. 信任可以升级：从被迫合作到默契配合
4. 信任可以崩塌：一个错误的信号可能导致信任崩塌

### 4.3 信任构建的写作模板

```
被迫合作：规则绑定，缺乏信任
金属项圈扣在两人脖子上。
你被打死了我也会死。齐夏说。
赵医生咬着牙：……我知道。

信号验证：通过事件建立信任
赵医生撒泼打滚，齐夏读懂了信号。
你做得很好。齐夏低声说。
赵医生愣住了——这是齐夏第一次夸他。

默契配合：不需言语的协作
下一回合，赵医生没有等齐夏的信号。
他主动丢弃绳子，拉开距离。
齐夏嘴角微扬——默契已经形成。

信任确认：信任的最终确认
游戏结束后，赵医生问：你怎么知道我是装的？
齐夏说：因为你的眼睛。
什么？
你在撒泼的时候，眼睛一直在看我。
```
